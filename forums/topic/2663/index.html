Quake 2 source released
Kinda indirectly related to ASM game programming, id has released the source for Quake 2 under GPL. The core is written in pure C (and VC project files) but it also has some monster asm modules for displaying of polygons (in ref_soft directory).

You can nab it at:
ftp://ftp.idsoftware.com/idstuff/source/quake2.zip
