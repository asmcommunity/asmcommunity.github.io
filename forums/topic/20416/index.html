Vbscript COM example trouble
the script engine needs type information to know the app's object model. IIRC the author of this little sample forgot to provide code to register the type info.

You can use comview to do this forgotten step:

1. start comview
2 select "Load type library"
3. load the .TLB file from the sample dir
4. button "register typelib" should be enabled. click it

Now the type lib is registered and the sample should run
