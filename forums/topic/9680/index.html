Push / Pushd Difference?
Hello, looking through EliCZ macros I saw that he use 'pushd' to push
parameters on the stack. I tried to look for it in the instruction set
help file. But I couldnt find it? what does it do exactly? maybe
I was just blind and didnt see it in there. Anyone?
Posted on 2002-12-19 15:45:28 by natas
pushd is a 4 byte push.
Used to explicitly tell the assembler the size of an object.

push eax ; assembler knows the size from the register.

pushd somewherelikealabel ; explicitly sizes the bytes to act on, when size is a question.
