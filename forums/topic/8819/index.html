Preffered loading address for a DLL
How to change preffered loading address for a DLL in MASM?
Posted on 2002-11-06 05:17:51 by Vaxon
Re: Preffered loading address for a DLL
\masm32\bin\link /nologo /DLL /SUBSYSTEM:WINDOWS /DEF:*.def /BASE:0xAAAAAAAA *.obj

AAAAAAAA is address you like in hex.
