:) :) :)

cx is a register.

ecx is a register. cx is part of ecx.

In Win32Asm, the structure being used for SIZE is SIZEL:

SIZEL STRUCT
x DWORD ?
y DWORD ?
SIZEL ENDS


This neatly avoids problems with the reserved keywords cx and size.

PS:
(SIZEL is already defined in Windows.inc)
