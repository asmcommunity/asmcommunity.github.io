AsmEdit V4.6 released (10-27-2002)
Hi all,

I've made many changes to AsmEdit.

Most have been from user request and some on my own.

I also fixed several rainy day minor bugs.

I've upgraded AsmEdit V4.5 to V4.6.

You can go to my Web Site to download the complete package (Masm32V1.zip),
or the service package (SerPk05.zip).

Both are quite large because of the Resource editor, Control builder,
and additional programs.

You can use the service package SerPk05 if you have previously
installed the complete package or SerPk04 dated 09-12-2002.

All of the source code is included.

My Web Site is at:
http://asmedit.massmind.org/

Read the 10-27-2002 section of the Readme.txt file before installing.
Don't forget to run the standalone Setup program after installing.

Check on the Bug Fix download packages, they are updated
as soon as I or someone finds a bug and I fix it.

Thanks,

Ewayne

Hi,

(AsmEdit - Build)

Error highlighting with multiple source projects.

Now when you assemble a project and if there are
errors and you left click to the left of the error
line in the Build edit window and if the file of the
error is not currently displayed AsmEdit will either
open that file or switch to the Tab of the opened
file and highlight the error.
(It will also work with .rc files)

Before AsmEdit would only highlight errors of the
displayed file.
