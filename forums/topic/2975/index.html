How can I make a function that has only ordinal number in DL
change your def file like this:
Library MyLib
Exports
Myfunc1 @1 NONAME
Myfunc2 @2 NONAME

where @1 is the ordinal number you want the function to have. NONAME (i dont know if necessary though) turns off compiler's name mangling (at least in C. dunno if needed here)
regards, Phueghy
