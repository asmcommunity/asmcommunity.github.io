*New* Alternative to Windows.inc
I have made a new main include file which gives the same functionality as Windows.inc. It is named Win32.inc so you can dload & test it without corrupting your setup. The windows.inc team might find this format easier to work with.

It has these characteristics

* All constants in alphabetical order
* All extra spaces removed
* All keywords capitalized
