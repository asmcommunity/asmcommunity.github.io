CyBerian, The following code is used in the dialogs WM_INITDIALOG message in your dialog proc. The hWin is the handle passed to the dialog proc. 

        invoke LoadIcon,hInstance,1
        mov hIcon, eax

        invoke SendMessage,hWin,WM_SETICON,1,hIcon
