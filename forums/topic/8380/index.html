[source] AsmMindWeb - hyperlinked notes editor
AsmMindWeb

The goal of this software is to help you manage your interlinked
thoughts, by interfacing with the web, your other software and
with your wonderful mind.

here is the source of AsmMindWeb, with some bugs:
- random error on write/read files (sharing problems?)
- minor flickering caret

what will find in source (keywords for search engine):
- richedit nightmare (EN_LINK structure, URL, text coloring)
- mouse hoover (kinda ...)
- listview used as list to handle history
- some comments inside

what you'll not find:
- well commented source
- optimized code
- ...

if someone wants to play with it, fix things, add new things
... please do and post your findings here so that all of us can enjoy

big thanks to:
Rohan Pall - for the ideea
KetilO - richedit parsing

... and of course this msgb users
