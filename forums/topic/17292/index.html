Hi MoonCrash,

You can always create the dialog in the dialog editor then use the Dilaog to Window tool to convert it to a CreateWindowEx type window. But no, there is no way to edit a window created with CreateWindowEx, after all there is no template to follow that can tell the software what the window consists of or looks like. It may be possible (actually thinking about it, it is quite easy) to capture a window that is running and displayed to a dialog then converting it to DLG format, I may one day consider adding this to Res2Dlg but not any time soon unfortuantely.
