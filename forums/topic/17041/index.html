Searching for IDE for MASM with integrated debugger
Ketil's RadASM has integrated debugging tools (VKim) and as well fully integrated breakpoints and int3 breakpoints for any JIT debugger. It also has an integrated Hex editor. You also have the option to run the executable inside a debugger (like Olly or GoBug) from the make menu. It would be difficult to have more extensive debugging tools than are available from within RadASM.
