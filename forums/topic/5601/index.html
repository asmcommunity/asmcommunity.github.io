#bloat ?
$BLOAT macro val:REQ

    _DATA segment

        db val DUP(0)

    _DATA ends

ENDM





$SLOTH macro val:REQ

    pushfd

    push eax

    xor eax,eax

    inc eax

    push edx

    xor edx,edx

    REPEAT val

        div eax

    ENDM

    pop edx

    pop eax

    popfd

ENDM
It's left as an exercise for the reader to find the values to these macros that exhibit the same size/timings as the HLL compilers that others in the office are using. :grin:
