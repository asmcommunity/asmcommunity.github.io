Re: macro help
Declare macro ID, Val, Min, Max
  str&ID byte   "&ID"
         Count = 16 - @SizeStr(&ID)
         byte   Count dup(0)
  val&ID dword  Val
  min&ID dword  Min
  max&ID dword  Max
endm
