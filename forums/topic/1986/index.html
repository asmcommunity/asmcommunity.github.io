It sounds like you need to learn the x86 instruction set. Art of Assembly is pretty good. Follow the links on this Webster page. You can find HTML, PDF and download versions of both the 16 and 32 bit versions of AOA. It also has links to other good MASM stuff:

http://webster.cs.ucr.edu/Page_asm/ArtOfAsm.html

There is a ton of info out there on the x86. The Intel manuals, Microsoft's MASM reference and programmer guides. Hutch's MASM32 package comes with the OPCODES.HLP file, that's not bad for a quick reference to each instruction.
