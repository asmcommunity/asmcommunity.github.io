Changing Font
Pradeepan,

To do what uyou are after depends a bit on what way you display the font, for the normal true type fonts, you normally use CreateFont which gets you the handle of the font which you then display with API calls like DrawText or alternatively, with a control you use the Sendmessage / WM_SETFONT combination.

What is important with font creation is to understand how the font is allocated and be sure to properly deallocate the font resource after its used otherwise you will get a resource leak..
