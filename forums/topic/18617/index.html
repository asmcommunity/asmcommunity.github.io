HLA v1.66 is now available
Hi All,

HLA v1.66 (Windows only, Linux is coming soon) is
now available on Webster at
http://webster.cs.ucr.edu/AsmTools/HLA/dnld.html

There are three major changes to the code:

1. I corrected a defect having to do with nested
records and fields that were arrays of other records.

2. I fixed the oversight of not allowing @here in
conditional jump instructions.

3. I've modified HLA.EXE so that it uses the path
to HLAPARSE.EXE as the base directory for the
hlalib and include directories if HLA cannot find
definitions for the hlainc and hlalib environment
variables (Windows only). This should eliminate
about 70% of the installation/1st time run problems.

Cheers,
Randy Hyde
