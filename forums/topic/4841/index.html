Resource Compiler!
If you want to use a resource file without having to resort to rsrc.rc just compile it using the command line:

\masm32\bin\rc ResourceFilename.rc

During the linking process just include the .res file like this:

\masm32\bin\link /SUBSYSTEM:WINDOWS filename.obj ResourceFilename.res
