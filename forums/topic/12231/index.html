Visual ASM V1.0.0.2
Hi all,

This is Visual ASM V1.0.0.2. The following additions/modifications have been done to the previous release:

1.Custom Syntax Files.(Go Tools-Options-General)

2.Separate Option for /OUT in the Make options. If this is not empty the Execute command looks for the file and the path specified here otherwise an executable file the name of which is the same with the project title which resides in the Project directory. (Request by Joshua-I hope this is what you meant)

3.Shortcut keys to more menu items.(Request by Joshua)

4.Remove TreeView items when they are not needed.(Request by Joshua)

5.You can now change Editor and Line Number fonts.

6.Custom Colors.

7.Various minor bug fixes and optimizations.

There is still more to be done. There is only one way they can be done though. You have to say...
