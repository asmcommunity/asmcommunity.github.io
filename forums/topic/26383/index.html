MASM Programers Reference
hadding, Showing signs of age, I prefer the book that came with MASM 6.00 as the macros have not changed much or at all. I have done most of my work out of the ALANG.HLP file that can be accessed from the MASM PWB but you can also convert it with the standard accessory program in MASM called HELPMAKE. This makes it into a text file that you can edit and remove the old DOS based stuff from it. The MASM32 help file has most of the useful stuff from ALANG.HLP in it for 32 bit but if you are after some of the older stuff, try converting ALANG.TXT to text format. Regards, hutch@pbq.com.au
