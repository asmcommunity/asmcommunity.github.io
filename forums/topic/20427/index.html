how can change the font color of listbox (masm32) thanks
WM_CTLCOLORLISTBOX:

  .if hBrush != 0

    invoke CreateSolidBrush, #RGB_COLOR#

    mov hBrush, eax

  .else

    mov eax, hBrush

  .endif

  ret

hBrush must be a global dword, and don't forget DeleteObject(hBrush) if you don't need them.
