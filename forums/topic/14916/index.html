Hey realvampire,

Why bother to use arccos when arctan could solve your problem?

cos x = adj/hypo
cos^-1 adj/hypo = x
sin x = opp/hypo
sin^-1 opp/hypo = x
tan x = opp/adj
tan^-1 opp/adj = x

cos^-1 = arccos, sin^-1 = arcsin, tan^-1 = arctan

b/a = gradient
gradient = tan x
