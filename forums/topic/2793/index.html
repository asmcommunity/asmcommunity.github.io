RadASM settings
The functionality that you don't like is added by an addin. Look in RadASM.INI for this:
[AddIns]

;FlipCase,x (1=Flip case, 2=Auto Show/Hide Output, 3=Both)

1=FlipCase.dll,1

2=DlgToWin.dll,1
The number after FlipCase.dll turns it on/off. I have it turned off above. Default is a 3 - that is what you should see.
