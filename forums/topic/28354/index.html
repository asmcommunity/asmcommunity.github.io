How to make enter key behave like tab key: I think you mean the default behavior of enter and tab in a dialog box? If this is true, you can do: 1) use a application defined dialog box class or 2) subclass your dialog box window when receiving WM_INITDIALOG message. In your window proc, check for WM_KEYDOWN/WM_KEYUP messages and change virtual key to before default processing.
