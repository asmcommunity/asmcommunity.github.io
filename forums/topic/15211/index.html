Hi, peansen, welcome to the board! :alright:

Under win32 we don't have access to the DOS calls, nor to the BIOS. We have to use the Win32 API instead. I recommend reading Iczelion's tutorials on windows programming, there you will find the basic skeleton for a Windows program, and much more.

If what you want is to compile your DOS programs from Masm32, there are some threads in this board on how t do that. Note that assembling a DOS program from Masm32 is not different at all from assembling with the DOS version of Masm.
