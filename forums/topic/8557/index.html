Multiple Paths
Hi _Shawn

RadASM setup does not handle multiple include / library paths.
There are several ways to work around this.

1. Set other paths in enviroment. (Win98 Autoexec.bat SET PATH=)
2. Give full path in the include and incluselib statements in your code.
3. Change project options.
