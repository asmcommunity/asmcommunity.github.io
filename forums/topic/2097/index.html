Main::How to Get Disk/Drive Information
The Drive type can be got with the API GetDriveType.

With GetDiskFreeSpace you can get
1) root path
2) sectors per cluster
3) bytes per sector
4) number of free clusters
5) address of total number of clusters

To use interrupt and read/write single sectors... Hmm... I'm not sure, but I think you should use VXD... Sorry but I do not know anything more about them.

Main::How to Get Disk/Drive Information
If you're progressive and thus on a NT-based windows, you can
get access to your drives and partitions with special strings to
CreateFile, without muckign around with VXDs.

To open a harddrive, you'd pass \\.\PHYSICALDRIVE0 to open the
first harddrive. If you want to open partitions, do it like \\.\C: .
