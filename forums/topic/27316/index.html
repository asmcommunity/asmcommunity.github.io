Getting the current column
Ive figured out how to get the current line the caret of a rich text box is on, but how do i get the column number?
Posted on 2001-03-20 18:09:00 by HeroInc
Getting the current column
This worked for me. 

.data
chrg     CHARRANGE<>
.code
invoke SendMessage,hRichEdit,EM_EXGETSEL,0,addr chrg
invoke SendMessage,hRichEdit,EM_LINEINDEX,-1,0
sub    chrg.cpMin,eax
chrg.cpMin now contains the column number
