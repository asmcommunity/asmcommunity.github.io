Hi _Shawn

RadASM is not designed to do this kind of stuff. But workarounds exist.

Here is one way to do it:

1. Dl the zip and unzip to a temp folder.
2. Copy the M32Lib.Rap to \Masm32\M32Lib

Win98:

3. Open the M32Lib project. Ignore the error message.
4. Select Project / Project Options and select the Debug option. This builds to console instesad of output. Win98 does not run .bat files to output for some strange reason.

NT, 2000, XP

3. Edit the Masm32\M32Lib\Make.Bat and remove 'pause'
4. Ignore error message when the M32Lib project is opened.
