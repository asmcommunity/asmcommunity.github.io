Hi rajmohan

The error you get indicate that ml.exe is not found. Check that it exist in the given path and that there is nothing wrong with the exe. There is also a switch in RadASM that displays the command string you are trying to execute in the Output window.

Use Project Options and change:

3,O,$B\ML.EXE .........

to:

3,OT,$B\ML.EXE .........
