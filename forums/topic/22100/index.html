Re: Getting Control Handles - Array?
If you have an array

array dd IDC_MYEDIT1, IDC_MYEDIT2, IDC_MYEDIT3

You can do something like


push edi
mov edi, 2
@@:
invoke GetDlgItem, hdlg, 
mov , eax
dec edi
js @B
pop edi


If I am not wrong.
