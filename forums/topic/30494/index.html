Fresh IDE source repository established.
Now Fresh IDE source code is hosted using Fossil version control system.
Regardless of its funny name, Fossil is really easy for use and small enough (1MB; single executable) to be used for assembly projects without an inferiority complex. ;)
Fossil engine, besides to version control, provides also bug tracker and wiki. Just click to the link below and you will end on the web site, created by Fossil engine.

The Fresh IDE repository is located here: Fresh IDE repository

In order to clone the repository for you, just type:
fossil clone http://chiselapp.com/user/johnfound/repository/FreshIDE/ Fresh.fossil


If someone wants to commit changes to the repository, just contact me with PM.

Regards.
