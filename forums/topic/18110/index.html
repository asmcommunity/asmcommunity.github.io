Hi all

New upload at the top.

- Notification messages to parent window.
- Select object(s): Select move from toolbox and leftclick on an object. Hold Ctrl to do a multiselect.
- Delete object(s): Select object(s) and press Delete.
- Size object: Select object, leftclick and drag a selectbox.
- Move object: Select object, hold Ctrl while leftclick and drag a selectbox.
