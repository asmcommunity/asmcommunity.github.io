warning LNK4089
That means you did not call any functions from user32.dll. It is just a warning not an error. A warning telling you that you have included user32.lib but did not use any functions from user32.dll
Posted on 2003-04-21 09:14:44 by roticv
warning LNK4089
Hi Red,

The error is just as roticv said, you can add /IGNORE:4089 to your link command line and it won't be displayed. It will only appear in "release" code as the linker will attempt to make the code smaller by not linking any dll's that are included but unreferenced.
