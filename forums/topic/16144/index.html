New search feature
Hi KetilO,

While programming in Delphi with GExperts addin installed, I got used to a neat feature it offers. It is a quick jump to next/previous occurence of a word the caret is at. You have a similar feature - just select what you want and press F3 - the editor will jump to the next occurence of the selected text. But it has two drawbacks:
1. you have to select the text first
2. the text is not treated as a word, but as a bunch of letters, that means when you select "ax", then pressing F3 will jump to the nearest eax, if it happens to be before the nearest ax

I'd like the quick search to get the word from the position of the caret and jump to it's nearest next/previous occurence.
