Re: How to add a bitmap resource to a Dll file?
Well, I'm using WinAsm Studio, which simplifies the whole thing... ;)

http://www.winasm.net

If you still want to use the command line tools to build the project, these are the commands:

ML /c /coff /Cp /nologo /I"C:\Masm32\Include" "DllResource.asm"

LINK /SUBSYSTEM:WINDOWS /RELEASE /VERSION:4.0 "/LIBPATH:C:\Masm32\Lib" /DLL "/DEF:DllResource.def" "DllResource.obj" "DllResource.res" "/OUT:DllResource.dll"


I'm assuming your MASM32 package is installed at the default location (C:\Masm32).

Hope that helps! :)
