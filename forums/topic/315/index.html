if you use a normal listview, there MUST be a notify also
in Win2K for a sel-change. If not, you probably have another
error somewhere else.

SDK states, that there will be some differences with the LVS_OWNERDATA style, where you must filter the LVN_ODSTATECHANGED notify message.
