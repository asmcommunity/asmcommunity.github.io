WinAsm Studio V3.0.1.4
Hi all,

WinAsm Studio V3.0.1.4 is available from my site

What's new
-Find Dialog: New option:"Direction-All" (Thanks andrew_k, DC)

-Rebar and docking windows color now behave properly when XP themes are used(Thanks PhoBos)

-Ln, Col & Sel indications are now displayed in real time(Thanks Marwin)

-Owner Drawn menus minor vertical change of caption drawing (Thanks PhoBos)

-New feature: Ctrl+F3 Smart Find finds next occurence of selected text(if any), otherwise next occurence of current word (Thanks PhoBos)

-Images (Thanks PhoBos) for:
CONVERT TO UPPERCASE
convert to lower case
Project-Add Files

-More improvements for Structure members intellisense

-Add Files button on main Toolbar (Thanks cmax)

-Run command works irrespective if there were errors or warnings during build(Thanks Qvasimodo)

-some issues when WS_EX_TRANSPARENT was used fixed (Thanks Qvasimodo).

-Procedures list does not display macros and structures any more (Thanks Qvasimodo).

-asm and inc files new optional style: Tab Indicators
From Tools-Options-Editor enable/disable and from Colors-Tooltips & Tab Indicators change color (Thanks Qvasimodo)

-Docking Windows Focus rectangle Improvements

-Relative file paths(Thanks Marwin)

-Various improvements

Improved Add-Ins Support:
1.WAM_GETCURRENTPROJECTINFO message always fills CURRENTPROJECTINFO structure members(Thanks Qvasimodo)

2.New member of CURRENTPROJECTINFO structure: pszProjectTitle(Thanks Qvasimodo)

3.WAM_ADDOPENEXISTINGFILE :EQU WM_USER+103 ;hMain,wParam=lpFileName, lParam=TRUE/FALSE(True to Add File to the current project, False to Open File external to the current project(Thanks Qvasimodo)


Zip file includes:
-New executable
-New version of CodeHi
-New version od WAAddIn.inc

Regards,

akyprian
