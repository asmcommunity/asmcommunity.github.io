NewWiz updated to 3.0.2.7
NewWiz version 1.3.0.0
Changes:
- New tab added to Project Properties, to configure the optional command line parameters. Will only appear on WA 3.0.2.7 or above, and only when the project type is 0, 2, 4 or 6.
- Additionally, you can keep and edit a history of command line parameters in the .wap file.
- Template support updated to use the new .wap file entries.
- Built-in default make commands for static library projects updated to use lib.exe instead of link.exe.
- Small bugfix when no templates were found.

NewWiz version 1.3.1.0
Changes:
- Now uses WAAddInLib.lib
- Fixed a bug in the templates support (%title% wasn't being parsed properly).
- Some parts of the code were rewritten.

Enjoy :)
