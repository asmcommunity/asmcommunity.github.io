resource editor
You can use ResEdit either as a stand alone
program or as an intergrated program for AsmEdit(
If you have the AsmEdit package installed).

It will create menus, resources, and dialogs.

It can also extract dialoges from binary files.
e.g. .exe, .dll, res files.

It will build a standard .rc file.

If you want to try it stand alone download the
ResEdit.zip package and update it with the FixRes.zip.

http://asmedit.massmind.org
