WM_KEYDOWN oddities
during a little test project for some kind of 3D game, I encountered that a standard windows keyboard is not always able to detect 3 keys in a row (on keyboard) pressed at the same time.

I used cursor keys to move my "ship" and space key for "fire". So finally I found that the keyboard does not detect the space key, if I'm pressing two cursor keys (e.g. right and down) already.

So I tried the CTRL key - everything is fine now!

If someone thinks I'm wrong, give me a hint :)
