Introducing: RadASM Direct AddIn
Well, I've been experimenting lately with a Delphi Direct-like feature for my own project. Since I had to reformat my machine (I am a MSDN Universal Subscriber and tried to install Visio 2002 from the latest shipment of CD's and didn't realize it required activation, so I opted out because I have a different drive for that -- I use removables -- and it locked me out of my whole system so I reformatted)... I gave up on RadASM it's too difficult to locate all the up-to-date files if you have to start over again.

Therefore, I created this, to keep people informed. I have a test website that I use for non-official purposes and it holds some files relevant for this. Here's what you do.


Download This File and then place it in the \Addins folder.

Modify your RadASM.INI. In the section, add a number, and then make it equal

&RadASM Direct,T,C:\RadASM\AddIns\RadASM Direct.exe

Save and start the IDE. Then choose "RadASM Direct" and you'll get some direct news. I'll maintain the file for a while until I can provide a tool for KetIO if he wants, otherwise I'll maintain as I can.

Lets make a change for the Addin settings. Currently that path is hard coded but it may not be the same for everyone, in the radasm.ini file, change this


6=&RadASM Direct,0,T,C:\RadASM\Addins\RadASM Direct.exe


to


6=&RadASM Direct,0,T,$D\RadASM Direct.exe

This way, whatever the path settings are for the configuration will take effect for the Addin, as well.
