using UNIONS in FASM???
Hi Kobra,
Yes....here is example 


struc Test

{

 .first dd 0

 .second dd 0

 virtual at .second

   .Third dd 0 

 end virtual

}


this would create a union between .second and .Third in the structure
here is a second example below, which breaks up .second into two .Third and .fourth 



struc Test

{

 .first dd 0

 .second dd 0

 virtual at .second

   .Third dw 0

   .Fourth dw 0 

 end virtual

}


anyway hope this helps...as i am new to fasm so ya enjoy
