greenant,
i can understand your apprehension about using ADO, it can look scary. But like i said, if you take the time to learn (some of) it, then it will more than pay off for you. SELECT

	COUNT(*),

	blah1,

	blah2,

	etc...

FROM

	yourTable,

	etc....
