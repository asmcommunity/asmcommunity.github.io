Ewayne,

This does not answer your question but I have found another viable backup solution that works very well. I bought a Pioneer DVR-108 DVD writer that came with the latest version of Nero which contains an application called InCD that will format a rewritable DVD so that you can use it like a normal disk.

I hooked some cheapie 4x rewritables and now all I have to do is format the DVD and leave it in the DVD writer and dump onto it whatever I like. Current rewritable media is 4.7 gig but I imagine the double layer ones will be available at a low cost over time which are over 8 gig.

The Pioneer DVR-108 / 109 writers seem to come at a reasonable price these days and the Nero software is very flexible once you get used to it. Something you will need to do if you go this way is to download the latest firmware patch from Pioneer and update the writer's firmware as it makes it capable of writing to a wider range of media.
