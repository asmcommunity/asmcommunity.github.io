Project browser addin
I have been thinking about a way to make managing groups a little easier and thought Drag&Drop would make life easier for me. With this addin you can move files around the project browser window by dragging and dropping them into existing groups. You still must use the group editor to add and enable groups. You can also drop a file onto the tab strip to open it.

Copy the DLL to the AddIns folder and enable it with Addin Manager.

Project browser addin
Sorry to the 3 people who downloaded it but I was thinking that another minor annoyance is adding files to a project. To improve this I have added functionality to the treeview that makes it a drop target by itself and will allow you to drop files directly into the current project. It will handle multiple file drops as well as shortcuts. The previous function of opening a file dropped into the treeview is no longer available but you can just drop it into the editor instead.

Please note that this is a "User must be sane and possess at least the intelligence of a rock" addin. I don't bother checking to make sure a project is open or that the browser is viewing the project tree so the results will be very unpredictable if you screw up.

I also neglected to preserve EDI in one callback, might have caused a problem in 9x.
