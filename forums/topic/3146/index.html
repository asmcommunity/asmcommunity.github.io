default buttons
Stan,

I believe that, for a dialog, the enter key is sent
as a BN_CLICKED message to the dialog's message
loop. You can set or change the default button
by using the DM_SETDEFID message. You may also
have to use the BM_SETSTYLE message to change the
border style of the previous default button.
