Hi Niu.

The CheckDlgButton and IsDlgButtonChecked API funtions need the handle to the parent window as the first parameter. When controls are inside a Group or a Picture object (which are containers) the parent is the Group or Picture object, not the window, so you have to pass the appropiate handle to those API functions. As Easy Code can nest controls inside other controls, it always sends the WM_COMMAND message to the owner window to make message processing easier. So, in order not to have problems, use the BM_GETCHECK and BM_SETCHECK messages through the SendMessage API function.
