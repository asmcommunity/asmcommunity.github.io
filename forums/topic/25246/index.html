Quick&Dirty Resize Lib
Recently, I need a resizable dialog, so I have written quickly resizing library.
Enjoy :P

P.S::
I have to warn you about Anchors, an anchor at least include (left + top) or (right + bottom)
for instance;
ANCHOR_LEFT + ANCHOR_RIGHT = Invalid
ANCHOR_BOTTOM + ANCHOR_TOP = Invalid
BUT !
ANCHOR_LEFT + ANCHOR_TOP = Valid
ANCHOR_LEFT + ANCHOR_RIGHT + ANCHOR_BOTTOM = Valid
ANCHOR_BOTTOM+ANCHOR_TOP + ANCHOR_LEFT = Valid
