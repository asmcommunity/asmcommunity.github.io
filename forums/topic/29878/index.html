NT-based Windows have never depended on (or included) DOS - unlike Win9x, the NTLDR bootloader loads the Windows kernel directly. 32bit NT versions include NTVDM, though, but even that doesn't come with a "full DOS".

64bit NT indeed don't include NTVDM anymore, since that would have required doing a full virtual machines; 64bit x86 "long mode" doesn't support 16bit tasks.
