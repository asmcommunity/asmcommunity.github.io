Re: How to set up a Tasm assembler?
Plopp Tasm and the file you would like to assemble in a directory called Tasm under C:\ so that it will be C:\Tasm\.
Type in under (Start Search) "cmd" without the quotes. This will open the command prompt.
Black Terminal opens. Type "cd\tasm\" (again no quotes)
Type the name of the tasm executable and read the help message on how to assemble files.
