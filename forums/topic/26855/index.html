How to make the Application have an Icon?
Simple - just make the first icon (in terms of number order) the icon you want for the prog. Then when you compile it becomes your program icon automatically (for explorer etc)... If you want the icon for the alt-tab and WindowText bar, use LoadIcon with ICON_BIG and ICON_SMALL as 2 calls - there are smaples of this around. James http://james.ezylink.net.au
