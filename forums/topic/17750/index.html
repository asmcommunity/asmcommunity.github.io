Using RadASM to edit html code
Hi all

If you like me used notepad to edit your html code then this is good news.

Using RadASM to edit html code.

Based on the work by pelaillo.

1. Copy html.ini and all files and folders in Html to C:\RadASM
2. Open RadASM.ini and change section:

Assembler=masm,fasm,tasm,nasm,hla,GoAsm,html
