Hi,
ok, found the solution...

.data
open db "open",0
name db "textfile.txt",0

invoke ShellExecute,hWnd,addr open,addr name,0,0,SW_SHOWNORMAL

Starting standart editor with textfile
Another solution: 



.data

textfile db "textfile.txt",0



.code

invoke ShellExecute,hWnd,0,offset textfile,0,0,0


The WIN32.HLP file also says that you should set the last parameter to zero if you are opening a file rather than an app, but in my windows using SW_* works Ok...
