Re: How to set up FASM to work with HLA

How do I set up FASM to work with HLA please? I want HLA to launch the FASM command-line program to compile the FASM files it creates, like MASM32, LINK are set up.


HLA -sf -fasm xxx.hla

This will compile and produce FASM-compatible source ("-sf") and then
use FASM to process the output ("-fasm").

Use HLA -? to see all the command line options.
Note that this code still runs MS-link to link the output.
