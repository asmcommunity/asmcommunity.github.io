GoAsm header project
I have completed translating the first set of header files for GoAsm and they are available at my site. The current version is 1.01 and though there are still a couple of large files to translate they are not called in the Windows.H chain and so can be done at my leisure, perhaps later this month.

Note that the syntax of these files are very GoAsm specific and will not work with other assemblers. The eventual goal is to provide a complete set of headers up to Windows Vista and as well a comprehensive set of switches and conditional compilation. The headers are designed to follow the naming conventions of C language headers in order to match the documentation at MSDN as closely as possible. The headers were translated from Pelle's C headers and the PSDK.
