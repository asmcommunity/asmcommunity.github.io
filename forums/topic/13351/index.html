Tooltips and "On Top" windows
You cannot set WS_EX_TOPMOST, only read it. To actually set topmost flag, use SetWindowPos(hWnd, HWND_TOPMOST, 0, 0, 0, 0, SWP_NOACTIVATE OR SWP_NOMOVE OR SWP_NOSIZE).
