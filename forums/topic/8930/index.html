fixed size windows?
Just define it in your CreateWindowEx call when you create the window how big it should be.
Don't use an Overlapped Window but define all what you need manually. Like WS_Caption etc...

Win32.hlp should give you some pointers to that ;)

(Basically you want to get rid of the WS_THICKFRAME Style.)
