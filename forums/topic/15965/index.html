Hi

In your first file you use the "include" statement

e.g. if you have

one.asm
two.asm
three.asm
etc

in one.asm you should have

Include two.asm
Include three.asm

You will also have intellisense help after typing "Include "

I hope this helps

PS. You method is ok (although not easy) but the length of command line is limited in MASM.
