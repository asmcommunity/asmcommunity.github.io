solar,

normally ML.EXE generates COFF format for win32 but from memory it can still produce OMF format as well. If you want to build 16 bit asm files, you will need to get the 16 bit linker from either Iczelion's site or from the microsoft site. The linker in MASM32 is a COFF linker and will not work with 16 bit files.
