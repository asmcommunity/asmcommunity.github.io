WinAsm Studio V3.0.0.0 Beta 4
Hi all,

This is WinAsm Studio V3.0.0.0 Beta 4

What's new:

1.You can Drag&Drop files to the WinAsm Studio Project Tree. If there is an opened project these are added to it, otherwise added to an automatically created new (exe) Project.(Thanks WinCC)

2.Find&Replace Dialog:
a)MessageBox informing "search complete" (Thanks JohnnyQuest)
b)Selected text from editor windows is automatically inserted as the Find What Text (Thanks maCo).
c)Combo to remember all "Find What" strings (Thanks akyprian )

3.Correct state of the Save File tool(Thanks WinCC).

4.Some corrections when saving Version Info (resource editor)

5.WinAsm Studio main window remembers its pos and size of last use (Thanks ET)

6.Selecting from the Project Tree, shows MDI Child maximized (Thanks defbase)

7.-New Project begins with single file maximixed (Thanks Qvasimodo)
-Add-Files Maximized (Thanks Qvasimodo)
-Open-Files Maximized (Thanks Qvasimodo)

8.Appropriate Menu/toolbar options not disabled Open Winasm --> New project --> Project / Add New RC (Thanks TheOne)

9.Some minor improvements
