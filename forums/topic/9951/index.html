> what is RVA?

RVA: Relative Virtual Address. You can find some info about this expression in
any document about PE files. Revise the Iczelion tutorial about PE files.

> where i can find information of how make/understand direct call to api or
> any dll i think

You need to know how a PE file imports functions from a DLL. This info you
can find it again in the Iczelion tutorial about the import sections in PE files.
