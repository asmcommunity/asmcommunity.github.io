Have a look at Iczelion's ODBC tutorials, and realize that it's no fun using ODBC - whether you code in a HLL or in Assembly.

www.sqlite.org has a C interface which is easy enough to use from assembly.

MySQL probably isn't suitable for what you want to do, since it requires setting up a server.
