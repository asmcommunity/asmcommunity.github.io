Re: How to use Visual Studio 2005 to edit .rc files?
It seems that you have to create a VC++ project and insert .rc file to it, to be able to edit resources. In older version of VS you could just open .rc file and edit it, now in VS2005 it is just in view mode if .rc file is opened without a project file. Sadly this adds quite much work for just to edit resources for asm programs :(

Well, it seems that you can create en empty Projectname.vcproj -file, which just includes rsrc.rc file and then double clicking that .vcproj file you can edit resources inside that rsrc.rc file. So it adds one file copy to make/edit resources with VS2005 to asm programs.

If anybody has a better solution, please tell me.
