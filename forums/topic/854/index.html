Dialog boxes
CreateDialogParam and DialogBoxParam create a window with an internal class name ("#32770" or similar). They allow you to use use RC files to store dialog box specs.

When creating a dialog with CreateWindowEx, you must call CreateWindowEx to place all the controls in your dialog. You can create each control individually, or replicate the resource parsing of the dialog functions.

Either way, CreateWindowEx or CreateDialogParam, you use IsDialogMessage to perform the conventional keyboard navigation.
