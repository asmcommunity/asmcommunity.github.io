Re: About Ascii Text Art ...
You need to write the equivalent of OCR (optical character recognition) software.
Theres an easy solution and a difficult one.
The difficult solution involves setting up a neural network of x * y pixels, and 'training' it to recognize various inputs as various characters.
This is the best solution because it can 'approximate' - it can say "yeah, thats pretty close to being a forward slash".

The easier solution is to hardcode various bitpatterns, and this means that it can only recognize 'perfect matches', and needs to be given some kind of 'error metric' in order to allow for 'approximation'.
