Re: swap pointers instead of storing array
Not sure what you want, but perhaps it is just this?
while (func(i))
{ 
    i++;
    foo = bar[(i-2) % 3] + bar[(i-1) % 3]
    bar = bar[(i-2) % 3] *bar[(i-1) % 3]
    boom = foo / boom[(i-1) % 3] - boom[(i-2) % 3]
}
