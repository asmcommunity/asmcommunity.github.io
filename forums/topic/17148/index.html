HLA v1.61 is now available
Hi All,
I've just uploaded HLA v1.61 to Webster. You can find it on the HLA download page at http://webster.cs.ucr.edu/AsmTools/HLA/dnld.html.

This version incorporates several minor defect corrections. It also includes a couple of new standard library routines: os.system (similar to the C stdlib "system" call) and fileio.exists (which checks to see if a file exists).

See the "modifications.h" file in the HLA v1.61 sources for more details.

Note that I've also updated the documentation....
