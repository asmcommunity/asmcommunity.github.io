STRUCT Problem ,,,,RGBTRIPLE
RGBTRIPLE is defined in WINDOWS.INC, which is provided in the MASM32 pack. It is defined as three bytes (as follows): 

RGBTRIPLE STRUCT
  rgbtBlue      BYTE      ?
  rgbtGreen     BYTE      ?
  rgbtRed       BYTE      ?
RGBTRIPLE ENDS
In order to declare an array of them in your code you need the following: 

include \masm32\include\windows.inc

.Data
pixels RGBTRIPLE 360 dup (<>)
That should then define an array of 360 RGBTRIPLE. Mirno
