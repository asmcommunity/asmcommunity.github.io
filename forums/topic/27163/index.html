How to display gif picures in my dialog ?
Hi LaptoniC, the e-mail addresses on my site aren't very reliable so you'd better use _thomas_@runbox.com as e-mail address. You can't use GIFs directly in a dialog like you would with bitmaps, you should either bitblt the bitmap on the dialog (handle the WM_PAINT message of the dialog like in my example with a normal window), or create an ownerdrawn static control (or button, can't remember), which you then draw by handling the WM_DRAWITEM message. If you have further questions just ask in this thread or mail me. Thomas
