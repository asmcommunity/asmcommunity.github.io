How to use AMD CodeAnalist with MASM
I've searched the board, but one thing i've found about subject is
this thread

Since i have no MS VS, but only masm, I use this in RadASM project option:

3,O,$B\ML.EXE /Zi /Fl /c /coff /Cp /nologo /I"$I",2

This works just fine for me with 1st and 2nd versions of CA

One throuble is no "normal" asm listing in CodeAnalist, when asm project is separated in some files. (just dissasembled list w/o labels, etc, that for me was enought)
But solution is easy -- use one asm file or some util that will "include" all files into one
