disease, The two files in the licence directory are the two Microsoft originals from respectively the WIN98DDK and the PLATFORMSDK and they establish your right to use the included Microsoft binaries to build assembler language programs. All of the software in the MASM32 package is copyright software and none of it is for sale, can be sold or included in any commercial package. Any software that you write using MASM32 is your personal property and you can do whatever you like with it including sell it for a profit. These restrictions are to protect the respective owners of the software and protect the project in general so that it remains true freeware. It has no association whatsoever with GPL or any other licencing system. Regards, hutch@pbq.com.au
