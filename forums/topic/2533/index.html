array of sockets
You could allocate an array of dwords, and then store socket handles in it. When you remove one, just set the handle to 0. When adding a new one, just find the first empty (0) position and stuff your handle there. Don't forget to check if you've passed the end of the array (no empty places left)! You could also use linked lists, but I don't think you really need it here.
