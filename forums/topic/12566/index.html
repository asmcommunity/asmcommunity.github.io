We cannot move memory to memory in a single instruction, thus you have to either use the stack 
push mem1

pop mem2

or use a free register 


mov eax,mem1

mov mem2,eax

PS there are some macro such as 
MOVmd	MACRO Var1, Var2

	push     Var2

	pop     Var1

ENDM
