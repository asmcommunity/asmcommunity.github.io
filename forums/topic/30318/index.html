Fresh 2.0.2 release.
There is a new release of Fresh. It contains bug fixes, new features and brand new API include files, containing almost all Win32 API functions prototypes ready for use with Fresh "Procedure argument hint" along with the project defined procedures.
The site for download is as usual: http://fresh.flatassembler.net

Also, there is new project in the "Related projects" section of the site: ImportsDB - the tool used to automatically build the new API includes. This utility is pretty flexible and can build include files in different formats, not only for FASM/Fresh.
