Size of C++ exe
Try the following:

1. Compile in release mode.
2. Do not use C++/C libraries
3. Remove the linking of C++/C libraries

A simple messagebox code be 1024 bytes.

Take a look at the following link:
http://www.catch22.org.uk/tuts/minexe.asp
