Use T in the command line to output it to the Output window, copy it and put it in a batch file... 

4,O[b]T[/b],$B\GORC.EXE /r,1

3,O[b]T[/b],$B\GoAsm.EXE,2

5,O[b]T[/b],$B\GoLink.EXE @$B\GFL.txt /entry Start ,3,4
