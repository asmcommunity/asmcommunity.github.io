Yes,you can link your obj. code with the resource file(.res). Only,you have play with the API functions:

Privalov's COFF output sample:

format MS COFF

extrn __imp__MessageBoxA@16:dword
MessageBox equ __imp__MessageBoxA@16
.
.
.

As usually : link /SUBSYSTEM:WINDOWS /LIBPATH:c:\masm32\lib fasmcoff.obj resource.res

I hope this will work for you.
