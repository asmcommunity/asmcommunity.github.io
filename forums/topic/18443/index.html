RadASM / Win98 / 16 bit code.
Hi all

This little stub solves a problem with 16 bit linker, or when running batch files with output to RadASM's output window on Win95, Win98 and Me.

Copy LnkStub.exe to \masm32\bin

Example link command:
5,O,$B\LnkStub.exe $B\DOSLNK.EXE,3
