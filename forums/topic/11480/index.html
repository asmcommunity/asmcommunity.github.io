Dear Registered VMware User:

We are proud to announce the availability of VMware(TM)
Workstation 4 Beta.

VMware Workstation 4 is a major release for us - a leap forward.
Please download, run it and give us your feedback!

The new functionality includes:

- OS Support and Choice. Microsoft Windows Server 2003;
Red Hat 8.0 and 8.1 beta, Advanced Server 2.1;
SuSE 8.0, 8.1, Enterprise Server 8; Mandrake 9.0.

- Full Debug Support. Full functionality of native debugging
in the CPU with support for user- and kernel-level debuggers,
including SoftICE.

- Snapshots. A point-in-time copy of the running system state,
saved to disk, supports easy rollback.

- Tabbing Between VMs. Tab between virtual machines in a quick
and natural way.

- Shared Folders Between Host and Guest OSes. The host folder of your
choice just gets mapped into the guest.

- VM Favorites. A browser-like Favorites List of VMs, remains
onscreen while you run.

- Drag and Drop Across Guest and Host. Drag and drop icons between
systems.

- Sound Blaster Audio Card. Listen to music with high fidelity with
support for a new industry-standard sound device.

- Wireless. A wireless bridge so you can use a wireless adapter to
access a network from a virtual machine. All IP protocols are
supported in bridged mode.

- DirectDraw Support. Run applications that use Direct Draw.

Again, we'd like your feedback, so please download and use
Workstation 4 Beta. We think you'll like it.
