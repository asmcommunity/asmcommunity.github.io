Re: DLL Stub
I've figured it out, and got it working!!  8)

The highlights:

; Get handle to real DLL loaded with LoadLibrary
push offset DLL
call LoadLibrary
mov handle,eax


then:

Function proc
push offset func1
push handle
call GetProcAddress
call eax
Function endp

etc...
