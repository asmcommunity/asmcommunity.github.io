Typhoon, Yeah, the listen API does just as it says.......it listens on whatever specified port. It is important to not that the port does NOT try to connect however; for that you use the connect API. If you want to be notified of incomming connections, specify FD_ACCEPT in WSAAsyncSelect.....then you can use the accept API :-) Good luck, *unknown* P.S.
