Somebody who has done it before should write a tutorial on the print spool.
I have done print stuff before (intercepting print jobs), and wouldn't be easy to write a tut on it, as a lot of security related stuff is involved, i.e. what you can do with the spooler APIs is governed by what security level you are running at on the domain. If you are running as admin on the print server, then you are sweet, but anything else can be a nightmare. And then you have to consider what rights you have with the printer, printer permissions are separate and distinct from domain permissions.....
