Well, the first thing I should think you'd want to check is that you save -twice- before you compile. Sometimes pressing the button on the toolbar works, sometimes it doesn't (usually because the mouse moves off the button just before you release the left mouse button). If you're sure that it's recompiling correctly, check for DDINVOKE SetDisplayMode. I've never tried it, but DirectX may automatically assign the DDSCL_EXCLUSIVE and DDSCL_FULLSCREEN flags to your lpDD if you tell it to change resolutions. If neither of those don't work, I'm stumped. :)
