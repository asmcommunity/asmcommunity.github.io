Re: How to Install RadAsm ?
Hi

The most important thing is to unzip the files so that RadASM.exe, RadASM.ini and masm.ini (and other assemblers) is found in the same directory. In your case move the files in C:\Program Files\RadAsm\Assembly\ to C:\Program Files\RadAsm\
Then use Option / Programming Language to add assemblers (masm.ini, tasm.ini etc.)
