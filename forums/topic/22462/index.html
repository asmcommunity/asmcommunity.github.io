Maybe you are not familiar with resource scripts. So i'll explain: you are able to set only main dialog font and its size, which will be default for all controls inside the dialog. So if you're trying to set edit control font independently, please use "CreateFont(Indirect)...SendMessage..WM_SETFONT" in your code :)
