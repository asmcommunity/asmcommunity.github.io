If you're interested in legally obtaining the software, there are three ways: 1) Buy a commercial license: Borland TASM 5.0 Commercial/Personal 2) If you're academic, get an academic license: Borland TASM 5.0 Academic 3) Auctions: Borland TASM 5.0 on EBay -- If up for Auction Borland TASM 5.0 on Yahoo -- If up for Auction 4) Software Resellers of Used and Vintage: Recycled Software Dot Com 5) A Simple Search on the Internet: If it can't be located on search engines, it's probably because people are respecting intellectual property or because the vendor is cracking down on illegal distributions of the software.
