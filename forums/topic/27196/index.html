How to make program?
If you know how to write a DOS exe file, use it as the DOS stub for the 32 bit PE file and then write you normal 32 bit file. In DOS it will run the DOS program, in 32 bit windows, it will run the windows code. You will need to check the linker's help screen to get the syntax to do it but it no big deal.
