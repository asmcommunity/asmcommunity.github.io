Re: Radio frequency triangulation
if the boat emits radio, with antennas you can measure timing-difference. Though, as radio travels too fast, that timing is measured in a tricky way.
With directed-antennas you can see the general direction from which waves come (highest input power). Rotation done by hand, or servo-motors.
