Visual ASM V1.0.0.4
Here is the new release of Visual ASM.

A lot has been done to meet most of your requests:

1.Output window hilites any line that user double clicks (different colors for Normal lines and lines refering to Errors). Double click to Go To Error. Output window cleared for new or open existing projects. Total Number Of Errors is displayed. If any errors exist, do not execute exe projects.

2.No need to edit the Make options any more-Only the command arguments (if you so wish).
So You must update the Make properties of all your existing projects.

3.After making a project all Errors are marked in the source files. All Errors cleared before new make.

4.Indent - Comment improved

5.Comment Blocks support.

6.User Selects binary path.

7.You can use Drag and Drop to change the order of your files in the project tree.(if they belong to the same category). Of course this influences the compilation of your Project.

8.Line Number Column Width Customizable.

9.Some minor bug fixes.
