ToolTipThief
Hi All,

I want to inform you that I have just finished working on a little tool (visit my homepage to download the Win32ASM sources) that allows the user to copy the content of the tooltips. The proggie intercepts the text from all the tooltips based on standard tooltips_class32 class (ANSI version only).

The coolest thing is that the system-wide WH_CALLWNDPROC hook is set without DLL.
