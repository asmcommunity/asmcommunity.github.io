RadAsm static box
The text you enter for the static control's caption is limited to 31 characters in Radasm. However you can change the caption when your program runs by 



invoke SetWindowText,hStatic,offset szLongStaticCaption

or 


invoke SendMessage,hStatic,WM_SETTEXT,NULL,offset szLongStaticCaption

the above bit of code will allow you to set the caption of the static control to anything you want. hStatic is the handle of the static control, and szLongStaticCaption is an arbitrarily long NULL terminated string.

to obtain the static controls handle, if it's part of a dialog box then do 



invoke GetDlgItem,hDlg,IDS_STATIC


hDlg is the handle of the dialig containing the static control
IDS_STATIC is the ID which you assigned to the static control

There's no way to set the caption in RadASM beyond 31 characters. However what you can do is manually edit the .rc file which RadASM produces to include the string you want.
