386 Book
Yup, I've had this for years. It's a great book and has tons of info. The systems programming section contains details on all kinds of "behind the scenes" stuff.

This should be required first reading for anyone interested in the 32-bit x86. Almost all of it still applies to the latest and greatest Pentium. They've made 'em faster, and used all sorts of tricks to do instructions in fewer clocks. They've added some bells and whistles like MMX, but the basic concept has changed very little.
