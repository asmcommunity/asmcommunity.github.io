Sounds like you may not have defined an entry point!
All standalone code (ie will compile to an exe) needs an entry point! 


.code



myentrypoint:   ; can be anything you like, I usually use start

                ; it is a lable so is just some text followed

                ; by a colon (':').



  ; Then your code is written here!



  invoke ExitProcess, eax

  ; End of the code!



end myentrypoint

                ; or "end start", or "end blah" depending on what

                ; you name your entry point


If it doesn't fix it, post the code!
