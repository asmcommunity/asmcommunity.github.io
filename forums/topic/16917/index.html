[tool] Gui4Cli - fast/easy interface for command line progra
Gui4Cli
link: Gui4Cli
desc: simple event driven language with which you can create and control GUIs
example: 


G4C INTRO_GUI

WINDOW #CENTER #CENTER  200 100 'My 1st gui'



xOnLoad

	guiOpen #this

	

xOnClose

	guiQuit #this



// -- ------------------------------------------------------

// Button declaration - Left  Top  Width  Height  Title

// --------------------------------------------------------



XBUTTON 30 35 145 40 'Click Me!'

	guiQuit #this


so, if you want to create a quick GUI it is a nice tool :alright:

btw, it has an interpreter that sits in the tray area and the code is the script in text form.
