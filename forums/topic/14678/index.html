HTML question
make a 16x16 (pixels) bitmap in paint . save it. open that in something like irfanview and save it as favicon.ico

upload that file (binary ofcourse) to the root of the site where you want it to appear (you can specify different icons for different directories if you want) and that's it.

Sometimes you need to refresh your browser's cache before it appears.
