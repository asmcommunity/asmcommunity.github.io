Cmax,
After doing some tracing of the 'display configuration'. I found two places
in the system registry were they are stored. They are called 'LogPixels'.
However, I only have Windows/2K so the values will differ on other versions
of windows.

.Registry
Assumed compatible with Windows/NT/2K/XP:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\FontDPI\LogPixels
Assumed compatible with Windows/9x:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\FontDPI\LogPixels

Assumed compatible with Windows/9x/NT/2K/XP:
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Hardware Profiles\Current\Software\Fonts\LogPixels

Other then the above registry entries I dont know how you can change it. And I dont know
if it will work to only change those values. But I tried to do it with 'ChangeDisplaySettings'
wich use the 'DEVMODE' struct, wich then include a setting for 'LogPixels'. This was done
without sucess. Well thats all I know a.t.m.
