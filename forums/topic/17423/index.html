Copy a single file from a defective hard drive
First add a secondary drive to the system so you have a place of destination since i doubt that OS registry will fit into a single bootable floppy...

Any available good HDD will do... I guess even an 100M or 500M old one

If you do not have one you should be able to find one at a seccond hand auction (just be sure is not very very old... i.e. is at least IDE :tongue:


After you have installed/added this new HDD as secondary master device:
Then boot from a floppy or CD with a command line version of Win9x and xcopy/copy the files to the new HDD.
