How to get the handle in a variable ?
Hi Red,

If your button is on a dialog box from a resource then use GetDlgItem to retreive it's handle, just pass the handle of the dialog and the ID of the button. For example in your dialogproc:
DialogProc proc hWnd:HWND, uMsg:UINT, wParam:WPARAM, lParam:LPARAM



.IF uMsg == WM_INITDIALOG

	mov eax,hWnd

	mov hDlg,eax

	invoke GetDlgItem,hDlg,MyButtonID

	mov hMyButton,eax



Dialog enp
