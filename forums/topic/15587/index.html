ELF linker for MASM OBJs
hi,

here?s the first version of a tool for linking from OBJ files into ELF files.

i coded it coz i noticed much peoples would like to code for linux, but dont like nasm and similars. with this tool, they can code in their favourite masm syntax(tasm in next release), compile and then link to working ELF files.

there?s a win32 and a linux version of the tool in the package.

the elf the tool output is very simple, and dont have sections. but run ok.

i included, as example, a socks4 server, for linux, coded in masm.
