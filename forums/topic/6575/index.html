Hi huh

First of all you need the masmApiWord.api file (included in full package). The file is plain text, so you can add your own words.

It is also possible to disable some wordlist functions: Option / Code Editor
- Wordlist Convert: Convert case according to wordlist.
- Prop. to Wordlist: Auto adds code properties (procs, constants, structs, macros,data) to the wordlist.
