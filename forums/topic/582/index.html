How to declare a string and initialize its value to a string
Afternoon, 4oh4.

If you want to display a (") then surround your declaration using (')s:
e.g. szFrameRate db 'Framerate "fps":',0

If you want to display a (') then surround your declaration using (")s:
e.g. szFrameRate db "Framerate 'fps':",0

... or use a combination:
e.g. szFrameRate db " 'Framerate' ",' "fps":',0
