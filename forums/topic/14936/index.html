There are links to many large programs here already. No one will post a large program here, this is a message board and not the place to post anything of that size, this is more for sniplets and small example apps. If you want to see some large programs look at RadASM or WinAsm, Toolbar Paint is approaching 6 thousand lines of code and is not quite complete yet. For the most part if you take the time to develop a program with thousands of lines of code you are not likely to make the source available, you may give away the software for free but the source stays yours. MASM and all of the other ASM packages are as capable as any other language to make a large program, but remember that a 600K Visual Basic program can sometimes only require a 10 or 20K in MASM so what is large in one language is probably not going to be as large in MASM.
