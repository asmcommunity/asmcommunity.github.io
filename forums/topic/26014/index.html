Batch file improvement proj
I am working on a way to have multiple "rc" type files in the same directory to avoid separate directories for each project.

Here is what I have come up with so far. I am trying to figure out how to integrate it with Qeditor.

:: copy an existing "rc" type file to rsrc.rc for processing
copy "%1" rsrc.rc /y
::
::
e:\masm32\bin\rc /v rsrc.rc
e:\masm32\bin\cvtres /machine:ix86 rsrc.res
pause
