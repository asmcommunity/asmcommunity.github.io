The GoLink manual is available online at http://www.jorgon.freeserve.co.uk/GolinkFrame.htm. For the most part in RadASM you would use the same command line syntax that was set up in the ini file. The problem is that golink does not use lib files so you have to have @CommandFiles to set up the dll's to search and the exports are handles in the DEF file which is another @ file.

At any rate check the manual or if you have a specific question about GoLink post it and I'll try to answer.
