How to define TreeView control in a resource file?
For controls that are not part of the standard set you use the CONTROL keyword and the class name. You also use this for custom controls. You might think about using RadASM, it has an excellent dialog editor that will do all of this for you and allows you to visually design your dialogs, it also support chinese characters and MASM32.

CONTROL "",{ID},"SysTreeView32",{STYLE},{x},{y},{cx},{cy},{EXSTYLE}
