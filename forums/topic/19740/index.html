HIDE request
Already there.

Project > Add to Link List

If you add a .lib file, it must be in the hlalib folder (if you wish to change that, edit the lib path environment in the .hpr file).

If you add a .obj file, it will consider the entire path.

The .libs being included are in the "Libraries" folder of the tree-view (you'll have to double-click it to open).

Several libs are not displayed there and are always assumed to be included (unless you change it manually), these are:

hlalib.lib
kernel32.lib
user32.lib
gdi32.lib
