Re: Time and Date Format

MMMMDDDDDYYYYYYY

Note that this is the American date format. The international date format is YYYYMMMMDDDDhh:mm



-What is the format of a "Time And Date" dWord stamp as encounted inside a PE?

This should be stored as a dword containing the number of milliseconds elapsed since Jan 1, 1970.


-How to access this (these) info(s) under Win?

If you have used FindFirstFile/FindNextFile to get a handle to the file, the info you require is in the WIN32_FIND_DATA struct. Or you can use GetFileInformationByHandle to fill a BY_HANDLE_FILE_INFORMATION struct (use CreateFile to get the handle).
