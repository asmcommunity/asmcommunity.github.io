Snippets library - offline version!
As you may know, I have a public asm snippet library on my site (here), on request of The Svin I created an offline version of it in .chm format. I will put it on my site in a few days but I'll post in here
now.

The chm has a table of content and a ful-text search option, there's no keyword index yet because I haven't found a way to generate meaningful keywords automatically.

The whole thing is generated automatically using some tools and php, which makes updating it easier.

I would like to say thanks to everyone who has contributed to the library so far. Everyone else, if you have time, browse your harddiskm, see if you got useful snippets and post them at my website.
