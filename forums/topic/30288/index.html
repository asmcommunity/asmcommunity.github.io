kinda,

Assembly language programming in most cases require assembly language compiler (to write simple programs you can use debug.exe, for an example of different case). There are many assemblers around, choose your poison.

Source files can be produced by various means, notepad.exe is one of them (any plain-text editor will do, even copy con: filename.asm).

If you're fluent with file type associations, you may run .asm source directly (well, it has to be compiled, then probably linked, then resulting executable run).
