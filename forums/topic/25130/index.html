Re: Searching a Scan (String in Files on Drive) Routine
There's two parts to this.

First is iterating over all files on the CD - FindFirstFile/FindNextFile, recursing on directories is the common way to do this.

The next part is searching the file for a text string. If you don't need wildcards, the fastest generic way is most likely a Boyer-Moore search.

If this is for some specific need, and not as a general tool for scanning CDs, you can make things MUCH faster by building an index of the files...
