AsmEdit V4.3 released (06-02-2002)
Hi,

I would like to thank the following for testing AsmEdit:

gscundiff
Beyond 2000
BitMan
Jerry
Sara

I've upgraded AsmEdit V4.2 to V4.3.
Below is a list of some of the new features.

1. You can now have up to 10 individual Bookmarks for each edit window.
2. Changed the properties list.
3. Fixed the memory leak while doing assemblies and links using Win9x, ME.
4. Cleared all heap resources on shut down.
5. Resource editor interface via the Tools Menu.
6. Visual aid when creating some window controls (Toolbar, Tab control, and Button)
the rest will come later via the Quick Code Toolbar.
7. The procedure name that your working in will display on the status bar.
8. Made several subtle changes from user input.
9. Made numerous fixes to AsmEdit and it's support programs.

You can go to my Web Site to download the complete package (Masm32V1.zip),
or the service package (SerPk02.zip).

Both are quite large because of the resource editor and additional programs.

You can use the service package if you have previously installed
the complete package dated 11-29-2001.

My Web Site is at:
http://asmedit.massmind.org/

Read the 05-31-2002 section of the Readme.txt file before installing.
Don't forget to run the standalone Setup program after installing.

Check on the Bug Fix download packages, they are updated
as soon as I or someone finds a bug and I fix it.

Thanks,

Ewayne
