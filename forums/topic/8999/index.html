Richedit - Getting font-handle
WM_GETFONT
wParam = 0; // not used, must be zero
lParam = 0; // not used, must be zero


Parameters

This message has no parameters.

Return Values

The return value is the handle of the font used by the control, or NULL if the control is using the system font.


You keep getting 0 as the return value as your control is using the system font. Simple as that.
