2.0.1.3
- Fixed a few bugs.
- Added Recent files to File menu.
- Added keyboard commands to RAEdit:
- Ctrl+Alt+ or AltGr+

- Home Jump between splitts
- End Open/Closes splitt

- Page Down Expand all procs

to jump to the unreferenced variable.
