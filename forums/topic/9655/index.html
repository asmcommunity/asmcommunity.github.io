[link] HT Editor - Hex editor
http://hte.sourceforge.net

HT is a file editor/viewer/analyzer for executables. The goal is to combine the low-level functionality of a debugger and the usability of IDEs. We aim towards a perfect hex-editing ability and support of the most important file formats.

The code analyser will have many advanced features that make understanding the analysed code much easier. These features include: call/jmp-tables, cross-references (xrefs), functions, special code wrappers for import-stubs, automatic label generation, powerful search functions
