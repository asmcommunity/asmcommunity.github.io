MsdnHelp keyword file for Intel's VTune Instruc32.chm
For the people that want to use the instruction set reference that comes with Intel's VTune package (instruc32.chm), I have made a table that you can import into the Keywords table in the MsdnHelp database. Don't forget to delete the old keywords first (helpfile ID's 111 and 109) and to add a new entry to the Files table.

If someone who doesn't have MS Access would like me to send them a complete DB, please PM me.
