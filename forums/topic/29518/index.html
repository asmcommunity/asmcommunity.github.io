FaJo XP File Security Extension

FaJo XP File Security Extension (called FSE) is an extension to the context menu provided in the Windows Explorer for files and directories. With this extension you are able to directly view and change the file access rights (NTFS only) and ownership. FSE therefore uses builtin features that are available on Windows 2000, XP (even HP Home).
