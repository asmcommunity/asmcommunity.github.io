RadASM 1.0.8 Released (01-14-2002)
RadASM V 1.0.8
-------------------------------------------------------------------

Warning: If you are updateing, make a backup.

Unzip RadASM.zip to C:\RadASM

Fixed Bugs and Improvements:
-------------------------------------------------------------------
-Fixed memory leak problems.
-Fixed exit RadASM problem.
-Fixed dbl click on error message in output window problem.
-Fixed negative control drawing problem.

New Options:
-------------------------------------------------------------------
-Set font for text files.
-Select release or debug project option.
-Project file groups. Assambly, Modules, Misc and Resorces.
Use project options to enable / disable,
-Added some support for tasm projects. Not completed yet.
Guess I need some help from a tasm user.

New Features:
-------------------------------------------------------------------
-Project browser now shows wich files are locked (red dot).
-External programs can be automaticly launced when a file with a
given file extention is opened or dbl clicked in project browser.
(See RadASM.ini and OpenFile.txt)
-Improved parameter handling on Tools menu. This can be used to
build and run masm32 and icztutes without having to create RadASM
projects first. (See RadASM.ini and ToolsMenu.txt).
-Make menu active when no project is open. Can be used to build
masm32 and icztutes.
-$R (path where RadASM is loaded) added to the path shorts.
-Support for project modules.
Add, remove and assemble the modules. The modules obj files are
linked into main project.
-FlipCase addin has some new features. Added by NEMO.
Auto open main asm file and auto show / hide toolbox. Disable
unwanted features in RadASM.ini
-vkim has improved his debug macros.

KetilO

Hi JimmyClif

In the Addin folder there is an ini file, Help.ini.
It must contain the path to your .col file.

Exsample:

C:\Programfiles\Microsoft Visual Studio\MSDN\2001APR\1033\Msdn110.col

Note that there must be no Cr+Lf at the end of the line.

You also need to activate the addin in RadASM.ini.
When this is done F1 should give you help on the word where the caret is.

New Features:
-------------------------------------------------------------------
