Whats RtlZeroMemory?
This function is called ZeroMemory in the windows api reference. *********************************************************** ZeroMemory The ZeroMemory function fills a block of memory with zeros. VOID ZeroMemory( PVOID Destination , // address of block to fill with DWORD Length // size, in bytes, of block to fill with zeros ); Parameters Destination Points to the starting address of the block of memory to fill with zeros. Length Specifies the size, in bytes, of the block of memory to fill with zeros. Return Values This function has no return value. See Also CopyMemory , FillMemory , MoveMemory
