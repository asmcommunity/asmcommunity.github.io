Sp1 and Sp2
SP2 doesn't require SP1, it will update plain XP just as well. If you want to save yourself some time, slipstream the SP2 service pack to your installation files. You can use a tool like NuHi's nLite to automate this process, as well as apply a number of nice tweaks.
