WDM Driver Tutorial
WDM appears to be NT driver + Plug-and-Play + Power management. It definitely is not COM. NT drivers are written in C style. You can use a C++ compiler, but you do not use or create C++ objects. Drivers use the Hardware Abstraction Layer (HAL), a set of C functions that hide the details of raw I/O for each platform (e.g., MIPS). Someone will need to discover how these functions are implemented before they can be used or bypassed.
