Hi Zebio,

If you use Ctrl+0...9 to set your bookmarks you can jump directly to a bookmark using that key. So for example if you are on a line and press Ctrl+0 the next time you press Ctrl+0 you will jump directly there bypassing all of the other bookmarks. You can also jump to a bookmark set this way from the "Edit" menu using the "Go to bookmark" selection and selecting which bookmark you wish to jump to. These are called "Named Bookmarks" in the help file. Named bookmarks are saved and will still be there when you reopen a project.
