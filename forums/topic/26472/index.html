how to find out size of file?
You can use GetFileSize, or GetFileInformationByHandle. Since you only want the file size, GetFileSize is probably the best option and also the easiest to get used to.
Posted on 2000-12-30 21:35:00 by Racso
how to find out size of file?
Both GetFileSize and GetFileInformationByHandle are good IF you already opened the file. If you don't want to open it just to check its size, use FindFirstFile
