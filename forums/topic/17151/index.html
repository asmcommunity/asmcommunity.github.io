I have been wondering how you've got it in your "smallest Add-In ever" :grin: :grin: :grin: :grin:

If you manage to get the project path (and name) then it will be easy to get the list with all project files !!!

You can get the full path project name from the recent projects in the ini file (it will be the most recent project)

Also, GetCurrentDirectory should give you the last used directory of the current process (this may be the project path, the path used to add, open or save files!)

I can also expose the project's full path name via a new message in a future release of WinAsm Studio.

Having got the project's full path name you get its files from the *.wap file or you can even enumarate all project tree items.
