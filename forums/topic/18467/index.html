HLA v1.65 is now Available
HLA v1.65 is now available from the HLA Downloads page (http://webster.cs.ucr.edu/AsmTools/HLA/dnld.html). There have been several bug fixes to the assembler, standard library routines, and to the w.hhf header file.

Also note that the Linux edition was brought up to date with HLA v1.65 and several problems in the HLA code and standard library were corrected.

This version was created to serve as the CD-ROM master for the second printing of "The Art of Assembly Language".

Enjoy!
