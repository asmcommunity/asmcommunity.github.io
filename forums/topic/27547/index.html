RGB Macro
Im using this RGB macro... RGB MACRO red, green, blue mov al, blue shl eax, 8 add al, green shl eax, 8 add al, red and eax, 0FFFFFFh ENDM My problem is that the variables im passing to it are dwords. Is there someway to change this macro to accept them?

RGB Macro
I think this will work: 

RGB MACRO red, green, blue
mov eax, blue
shl eax, 8
add eax, green
shl eax, 8
add eax, red
ENDM
(and eax, 0FFFFFFh can be left out then) Thomas This message was edited by Thomas, on 4/5/2001 5:48:37 PM
