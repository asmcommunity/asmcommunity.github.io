Re: 8086 programming help
You can grab PDF copies of the latest Intel Processor Manuals for a reference.

In conjunction, find an assembler/disassembler combination that you enjoy using, and use the disassembler to dump out what the assembler generates. Study and cross-reference the disassembly opcodes with the aforementioned manuals.

PS: Use an assembler that can produce flat binaries, like NASM, FASM or YASM.
