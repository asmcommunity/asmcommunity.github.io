What is wrong with WIN32_FIND_DATA ?
All the structures and equates in w.hhf are in the w namespace so the structures and equtaes you use must be prefixed with "w.". 

 

program fff; // (FindFirstFile) 



#include( "w.hhf" ) 



var 

FindFileData: w.WIN32_FIND_DATA; 

endvar; 



begin fff;
