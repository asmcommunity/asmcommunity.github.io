Guy,

There are a couple of ways to get the result you are after, if the apps are dialog apps, you can open the RC file and change the font size larger so the interface is larger.

With an interface that uses CreateWindowEx(), you basically dynamically size the app at start so its a percentage of the screen in width and height. If things like buttons and other controls are too small, you will have to manually change the interface size of the controls.
