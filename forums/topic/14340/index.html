HLA v1.54 is now Available
HLA v1.54 is now available on Webster.
http://webster.cs.ucr.edu/Page_hla/0_hla_dnld.html

Most of the changes are relatively minor (couple of include file changes and additions, couple of fixes to Linux stdlib routines, lots more example code in the WPA subdirectory).

This release is the version that is shipping on the CD-ROM for the published edition of "The Art of Assembly Language" from No Starch Press.
