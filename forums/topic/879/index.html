Titan,

As long as you have not changed the batch files that are in the BIN dir, all you need to ensure is that QE is in the SAME directory as the main source file for the project that you are building.

QE always has a "current" directory and this cn be changed by the "Set Current Directory" option on the file menu. All that is happening is that if it is NOT in the same directory as the main source file, the file is not available to ML/LINK as it is written in the batch files.
