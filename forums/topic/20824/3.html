I want to suggest 3 things to improve RadASM:
1. Check if an application instance is running before you compile and run it (GO button). It happens many times, that I forgot a running app and I get an error message after a large compile time.
2. Before a system shutdown or an user logoff, save the unsaved source files. If you forget to do it, RadASM has to alert you.
3. An option to minimize RadASM after successful compilation and before execution of the app.
