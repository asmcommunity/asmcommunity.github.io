I use this: 



        invoke SetWindowPos, hCtl, 0, 0, 0, 0, 0, ; redraw window

                             SWP_NOSIZE or SWP_NOMOVE or \

                             SWP_NOZORDER or SWP_NOACTIVATE or \

                             SWP_FRAMECHANGED
