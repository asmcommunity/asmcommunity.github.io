GoAsm - a new assembler
First official release of GoAsm (Version 0.16 beta)

This is a new assembler written by me which works under the Windows operating system and which creates COFF object files.
I have tried to keep its syntax simple with the main aim of clarity and consistency.
It has a number of extensions to make programming for Win32 easier.
I would welcome bug reports and comments to jg@jgnet.co.uk.

To download GoAsm click here

It is a sister program to my GoBug debugger and to my resource compiler
