CreateNamedPipe()

Windows Me/98/95: Named pipes cannot be created.


But


Windows Me/98/95: Pipe names cannot include a colon. Therefore, if this pipe will be used from a Windows Me/98/95 client, do not include a colon in the name.


You cannot create a named pipe from Win 9x but you can access to a named pipe from Win 9x. The pipe is created on a server, and you, over the network, access to the pipe from your win9x client
