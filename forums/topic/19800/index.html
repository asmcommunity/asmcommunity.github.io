btw, What are pull ups? :-D :-D :-D :-D


when you have an digital electronic circuit, you can never let input ports "float", leaving it connected to nothing (that will cause problems, because of noise and potential differences and such). for example, you have a switch connected to such a port, and when you don't press the switch it would float.

to fix this, you can add a resistor (about 1k) to the input port, that is connected directly to the vcc. when you press the button, the input port will be connected to the ground and a little current will flow from the vcc to the gnd through the resistor. in this way, a port is never "floating". this is called a pull-up :)

schematic of the circuit: 



    ___ VCC

     |

    | | 

    | | R

    | |

     |

i ---|

     |

        /

      /`  s

     |

     |

    _|_ GND


where i is the input port, s the switch and R about 1k (not too low, to make the current low, not too high, to avoid noise).
