Can you #define constants like in C in ASM
YourPhoneNumber EQU 9762358 ; this is number

MyPhoneNumber EQU <976-WETT> ; this is text

; these are both constant
; (ie re-definition is an error, unless it's the same, iirc)

; you can also check if they have been defined:
IFDEF YourPhoneNumber
MyPhoneNumber EQU 9761234
ENDIF

; you can also check if they have not been defined:
IFNDEF YourPhoneNumber
YourPhoneNumber EQU 9761234
ENDIF
