Jumping in macros
Create a label and set it as LOCAL. 
gcd MACRO reg1, reg2

	LOCAL _1, _2, _3, _4



	neg reg1

	je _3

  _1:	neg reg1

	xchg reg1, reg2

  _2:	sub reg1, reg2

	jg _2

	jne _1

  _3:	add reg1, reg2

	jne _4

	inc reg1

  _4:

ENDM
Now we have a new instruction.
