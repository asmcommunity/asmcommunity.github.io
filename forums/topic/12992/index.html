Center text in an Edit box
Create the edit box with the ES_CENTER style. Pay attention that if you are using Win95 it only works if the edit box is multiline.

From MSDN:
ES_CENTER
Windows 98/Me, Windows 2000/XP: Centers text in a single-line or multiline edit control.
Windows 95, Windows NT 4.0 and earlier: Centers text in a multiline edit control.
