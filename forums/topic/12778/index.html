Edit Control
I have created a application using DialogBoxParam the Main window contains an Edit field. After

the user has typed in the data and pressed the command button a messagebox is show containing

the data from EDIT field. I want to check if the user has modified the Text in the edit feild how can i do

this??

Edit Control
EM_SETMODIFY
EM_GETMODIFY

to clear the modify flag :

invoke SendMessage,hEdit,EM_SETMODIFY,FALSE,NULL

to check to see if it has been modified since cleared :

invoke SendMessage,hEdit,EM_GETMODIFY,NULL,NULL

eax will contain TRUE if the text has been modified
