PoASM Wizard for Pelles C
I while back I posted a Dialog As Main PoASM wizard on the PellesC forum, apparently it got deleted after a certain amount of time. A user there posted a thread recently asking about it and it sparked me to go ahead and rewrite it (since I had since lost the source). The new version is available as an attachment to this thread and on my website ( http://www.codegurus.org/~bkeller ) under the projects section.

Wizard Supports:
- Dialog As Main Application
- Win32 SDI Application
- Win32 Console Application
- Win32 Dynamic Libraries
- Win32 Static Libraries
