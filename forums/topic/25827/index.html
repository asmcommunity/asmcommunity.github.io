Re: invoke EndDialog,hWnd,-1 what is the mean of -1
From MSDN

BOOL EndDialog(          HWND hDlg,
    INT_PTR nResult
);
Parameters

hDlg
Handle to the dialog box to be destroyed.
nResult
Specifies the value to be returned to the application from the function that created the dialog box.
Return Value

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.


Remember that -1 = 0FFFFFFFFh
