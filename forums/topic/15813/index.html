HLA v1.59 Is Now Available On Webster
HLA v1.59 is now available on the HLA Downloads Page
http://webster.cs.ucr.edu/Page_hla/0_hla_dnld.html

This is a maintenance release with the following improvements:

Corrected a couple of problems related to "extern" output for VMTs (in classes).
Corrected a defect in forward referenced types in pointer declarations.
Corrected a problem with the "-i:path" command line option.
Under Linux, there was some Windows "exit" code emitted in a couple of places,
corrected this.

Corrected a defect in a couple of constant-type conversions.

Added the ability to specify type coercions of the form "(type <typename> expr)"
for constants as well as memory locations.

Added the "-r:responseFile" command-line option.

Cheers,
