VC6 as IDE: Syntax highlighting possible?
You can have limited syntax highlighting (Asm keywords in blue). In the same directory as msdev.exe, use a Text editor to create a file named "usertype.dat". Add the ASM keywords to the file, one keyword per line. You can find the list of the ASM keywords in the MASM manual. Save the file, exit, and then restart Visual Studio. When you load an asm file, set the language to C/C++ (right click in the editor, "Properties").
To highlight the comment, follow the semi-colon with '//'. For example

;// comment

For multiple comment lines :

COMMENT /*
comment1
comment2
*/

Good luck!
