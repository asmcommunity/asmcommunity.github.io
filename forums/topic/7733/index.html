RadASM is really just a very advanced source code editor, to assemble your code you still need NASM, TASM, FASM or MASM. I suggest downloading the Masm32 package, as it's the most used and very complete. You can find the download URL in the FAQ and MASM32 forums.

PS: ml.exe is the MASM32 assembler.
