RadASM 2.0.3.2 testing
Hi all

Help me test RadASM 2.0.3.2 for bugs.

Whats new:

- Integrated hex editor (remove HexVU addin).
- Fixed Versioninfo bug (only partially updated the .rc file).
- Made changes to how dialog editor saves controls in the .rc file.
-- All controls saved as type CONTROL.
-- Important: If changing old dialogs then listboxes might need Notify to be set to TRUE.
- Added Notify option to several controls.
- Several minor changes / bugfixes.

KetilO

08-15-2003,3 dl,Fixed code properties bug reported by Morris, also fixed code property bug where RadASM would GPF if first word in a file was a code property word.
08-17-2003,26 dl,Fixed a selection paint bug. Added more controls (HotKey, Pager, ReBar).
08-17-2003,16 dl,Fixed a controls caption bug. A controls caption is now updated while typing.
08-18-2003,28 dl,Added tab action option on code complete list. Fixed an EN_SELCHAGE bug.
08-19-2003,31 dl,Added new styles to toolbar and statusbar. Made some changes to RadASM's toolbars. Added position in statusbar on hex editor. Did some work on rebar demo.
08-20-2003,31 dl,Added more functions to info tool (dialog and hex edit). Fixed a bug in dialog edit where a window handle could be resized to the size of the dialog.
08-21-2003,32 dl,Fixed api tooltip bug. Fixed toolbox paint bug. Made changes to RadASM toolbars colormapping.
08-25-2003,62 dl,Improved hla support. Made the release.

IMPORTANT:
You must previously have installed 2.0.3.1

That's the older format for the [code] section, replace it with this:

As Ketil had said above you had to be running RadASM 2031 to be able to use this version. So you have to use the 2031 ini file format. There were changes from the ealier versions.

[CodeMacro]
;KeyWord,InsertBeforeActiveLine,InsertAfterActiveLine
;{I}=Copy indent from keyword line
;{C}=Cr
;{S}=Space
;{T}=Tab
;{$}=Copy Proc/Struct Name
;The macros are set up to assume AutoIndent on
1=.if,{T},{C}{I}.endif
2=.else,{T},
3=.elseif,{T},
4=.while,{T},{C}{I}.endw
5=$ macro,{T},{C}{I}endm
6=$ proc,{T}{C}{I}{T},{C}{I}{T}ret{C}{C}{I}{$} endp
7=$ struct,{T},{C}{I}{$} ends
8=$ struc,{T},{C}{I}{$} ends

[Code]
Code=$ endp,$ proc
Const={C},$ equ
Data={C},$ db,$ dw,$ dd,$ dq,$ df,$ dt,$ byte,$ word,$ dword,$ qword,$ <?>,$ {?},$ <>
Macro=endm,$ macro
Struct=$ ends,$ struct,$ struc

[CodeBlock]
1=$ proc,$ endp,,,6
2=.if,.endif,.elseif,.else,0
3=.while,.endw,.break,,0
4=$ struct,$ ends,,,6
5=$ struc,$ ends,,,6
6=$ macro,endm,,,14
7=.data,.*,,,0
8=.data?,.*,,,0
9=.const,.*,,,0
