how i can create a DLL with TASM?
Ok, here's a short example which should get you up and running. SAMPLE.ASM 

.Code
DllEntry PROC hInstDLL:DWORD, reason:DWORD, reserved:DWORD
        
        ret
DllEntry ENDP

SomeProc PROC
        ret
SomeProc ENDP

PUBLICDLL SomeProc

End DllEntry
SAMPLE.DEF 

LIBRARY Sample
EXPORTS
SomeProc
Hope that helps.
