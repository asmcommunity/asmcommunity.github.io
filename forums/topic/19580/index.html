Driver development with GoLink 0.25.3
This is for users of GoLink.

I was asked whether I could add the necessary command line switches to to permit the creation of drivers (with the "sys" extension).

GoLink 0.25.3 which is available from www.GoDevTool.com therefore adds the following command line switches:-

/driver make driver
/wdm make a Windows Driver Model driver
/uponly set the IMAGE_FILE_UP_SYSTEM_ONLY flag (this indicates to Windows that the file ought not to be run on multi-processor systems)
