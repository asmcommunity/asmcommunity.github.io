DDE, Dynamic Data Exchange is what you want to look at if you want to do it right:

http://msdn.microsoft.com/library/default.asp?url=/library/en-us/winui/winui/windowsuserinterface/dataexchange/dynamicdataexchange/aboutdynamicdataexchange.asp

Microsoft word has a DDE interface that allows you to communicate directly with the application. You can find documentation for the interface at MSDN. In effect you open a DDE channel and send DDE commands to MS Word directly.
