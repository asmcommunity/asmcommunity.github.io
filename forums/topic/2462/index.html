ListView Controls as a Resource
This is from a recent project:

CONTROL "",404,"SysListView32",LVS_REPORT | WS_CHILD | WS_VISIBLE | LVS_SINGLESEL,2,62,264,107,WS_EX_CLIENTEDGE

404=dlg id
SysListView32=type of control
LVS_REPORT-->LVS_SINGLESEL=window styles
2=x
62=y
264=width
107=height
WS_EX_CLIENTEDGE=extended window style

Of course you can edit that for whatever window styles and extended styles you need. As for your second question I'd suggest getting a copy of the resource compiler help file. You should be able to find it by googling for "rc.hlp".
