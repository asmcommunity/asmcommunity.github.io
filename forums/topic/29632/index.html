Book Enhancements
I would like to announce that the ASM Book is back to using MediaWiki: http://www.asmcommunity.net/book/

With better forum integration between MediaWiki and SMF these days, and SMF 2.0 on the horizon, deprecating the custom book integration seemed like an obvious choice.

The following enhancements have been integrated into this newer Book system:

1.) If you are logged into this forum, and are apart of the "Code Warrior" (100+ post) group, you will automatically be logged in to the Book as well... thus making editing a more seamless task.

2.) A MediaWiki extension is installed that allows for code syntax highlighting, e.g. "odometer.c" from the Number Systems page.

Now, that being said, switching back to MediaWiki is going to incur some small growing pains. There are some pages that will have to be updated to display properly again. Please allow some time for (or assist in) the updating of these pages.
