link error: not valid library
I just removed the C: from the paths of the include and library file and it build correctly first time. I used the default setup with MASM32. 


    .386

    .model flat, stdcall 

    option casemap:none

    

    include \masm32\include\kernel32.inc 

    includelib \masm32\lib\kernel32.lib 

    .data 

    .code 

    start: 

    invoke ExitProcess,0

    end start

Just check if you are using the right linker or alternatively if the path to the library and include file are correct.
