GoLink now 64-bit
GoLink is now able to take one or more 64-bit object files and a res file (if used) and make a 64-bit executable which can be run on x86-64 processors.? No lib files are needed since GoLink looks inside the DLLs themselves to see which APIs they export.?
GoLink is at Version 0.26.0 beta and can be downloaded from here.? This link will also work for updates until the version is officially published.? Its early days yet so all bug reports would be appreciated.
