Resource compiler
If you installed Masm32 in its default location (C:\Masm32), then the resource compiler is located at C:\masm32\Bin\Rc.exe, to make the Tutorial 8-1 (menu) you can use the following steps in a DOS Box:

Cd C:\masm32\Icztutes\Tute08-1\
Ml /c /coff /Cp Menu.asm
Rc Rsrc.rc
Link /SUBSYSTEM:WINDOWS /LIBPATH:C:\Masm32\Lib Menu.obj Rsrc.res

HTH
