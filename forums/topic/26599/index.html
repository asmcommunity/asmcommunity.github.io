How to display bitmap on dialogbox
you can easily display a bitmap in your dialog without using any dlls. just put something like this in your resource script: #define BITMAP_1 1 BITMAP_1 BITMAP "mybitmap.bmp" and use a static control to display the bitmap: CONTROL BITMAP_1, -1, "static", SS_BITMAP | WS_CHILD | WS_VISIBLE, 4, 4, 0, 0 what's important is to use the ss_bitmap style and use the bitmap identifier (bitmap_1 in this case).
