Re: Rebar: put multi ctrls in a RebarBand ?
Create a child window (blank no borders) and this will be the child of the REBARBANDINFO.  Now add as many children to the blank window as you need...

IE

REBARBANDINFO (note only one child) ->
     Blank child window ->
          STATIC
          COMBOBOX
          BUTTON
