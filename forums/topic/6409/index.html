the X-includes - Bugreport !
Hi !

Someone found an error in the definitions of the IDirectDrawX-Interfaces. It wasn't possible to call for example the TestCooperativeLevel-function.

Of course I've corrected that and updated the X-includes.

You can make the necessary changes also by yourself. Just edit the includefile DDraw.Def and search for 



IDirectDrawX	STRUCT

	_vtIDirectDraw IDirectDrawX

IDirectDrawX	ENDS


where X can be 2, 3, 4 or 7. Please put the versionnumber behind the _vtIDirectDraw statement like 



IDirectDrawX	STRUCT

	_vtIDirectDrawX IDirectDrawX

IDirectDrawX	ENDS


and it's done !
