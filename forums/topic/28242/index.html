Use the dwtoa function in the MASM32 library. For example, 

.DATA
buffer BYTE 11 DUP (0)

.CODE

INVOKE  dwtoa, eax, ADDR buffer
; display eax
INVOKE  MessageBox, NULL, ADDR buffer, NULL, MB_OK
