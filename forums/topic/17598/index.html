"cut" row from file (LINUX)
cut works within a line of text, so it alone cannot do it (cut -d \n -f 2,3,7 for example won't work).

Try cat my_file | tr "\n" "\t" | cut -d \t -f 2,3,7 | tr "\t" "\n"
