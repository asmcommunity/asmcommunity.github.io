KSmake ready for general testing
KSmake version 0.15

See the KSmakeDoc.txt for more information on how to use it and command-line flags.

4.24.04
Fixed makefile bug where you needed a CR/LF at the end of the file. Now you no longer need the CR/LF.

Made it so undefined macros in the makefile are ignored along with any possible extensions they may have.
So if you have an undefined macro that looks like: " $(mymacro).rc ... " KSmake will not copy the ".rc" to the commandline. This way, you can build generic makefiles and as the project grows, you can simply add macros to the macro section without having to alter each section.
