using mouse wheel
Hey I had a need for the use of a mouse-wheel one time as well. It is very easy, just make use of a WM_MOUSEWHEEL message. Like so: 
.if uMsg==WM_MOUSEWHEEL
	mov	ax, word ptr 
	test	ax, ax
	jg	@@up	; Mouse Wheel is being rolled forward
	jl	@@down	; Mouse Wheel is being rolled backward
.endif
