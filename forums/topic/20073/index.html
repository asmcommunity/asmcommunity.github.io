New work release of Fresh (1.1.B)
Hi.
Now, the next work version of Fresh are available for download from the Fresh download page.
The main new feature is that Fresh is able now to directly run Linux applications in Windows environment.

To test this feature, you need the Linux emulator "Line" from http://line.sourceforge.net or http://sourceforge.net/projects/line/

Simply unzip Line in some folder, copy there cygwin.dll and set the path to the "line.exe" in the Fresh "Options|External applications" dialog.

There is a simple "hello world" example in directory: "%Fresh%/examples/ELFExe/". Load it in Fresh and simply press F9 key. 8)

Any suggestions and opinions are highly appreciated.
