Those addons could also be used with goAsm,hla,nasm and tasm. If you use FASM, simply add brackets to each API function.
FASM examples:
fasmApiCall.API -> ,dwFlags,th32ProcessID
fasmApiConst.API -> 6,NULL,CREATE_SUSPENDED
