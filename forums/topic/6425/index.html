SOCKET object (OOP model)
I made this object as a test of the OOP model created by Thomas & NaN.
It does all the basic TCP/IP winsock stuff, plus it has some extra functionality built in like receiving data one line at a time if required. The object file is compatible with NaN's OOP2HTML utility, in fact the output HTML help file is included in the zip.
I tested it pretty well, but I wouldn't bet on it being bug-free yet.

Ok I uploaded a new version, the new classname is CSOCKET. Thx for the tip, grv.
