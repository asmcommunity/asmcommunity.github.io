FASM and FreeBSD
In the ELF header of the Linux version you should replace the line:
db 0x7F,'ELF',1,1,1
with
db 0x7F,'ELF',1,1,1,3
because without this value Linux emulation in FreeBSD won't be turned on.
This fix is thanks to Peter van Dijk, version 1.41 will include it.
