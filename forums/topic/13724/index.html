Visual ASM V1.0.0.9(Bug Fix-Samples included)
Hi all,

For those who already downloaded, please do again. Small bug for tools-options-keywords fixed. Also 4 sample projects are included.8 June 2003


This is the new release of Visual ASM.

Please send your comments, suggestions, bug reports so that this IDE become better:

What's new:

1.Add External Tools(Tools Manager not implemented yet-to add/remove
tools go to the VisualASM.ini in the section and make the
necessary changes).

2.Recent Projects in File Menu(Thanks Manos).

3.Option to show open dialog on startup(Thanks Bi_Dark).

4.Change Keywords and their attributes from Tools,Options,Keywords(Thanks Manos).

5.Only the File used recently is shown after opening a project(Thanks Manos).


Bug Fixes

1.Bug Fix: WinNT and WinXP Bug not saving colors in ini(Thanks ??? I can't remember).

2.Bug Fix: Font Charset was not saved in NT(Thanks Dmitry).

3.From Project Explorer child did not open if item was grey (Thanks Manos).

4.XP bug. Double clicking on an errored line was forwarding us to a wrong place.

5.Small bug for first time cut or copy: paste was enabled only after selection change(Thanks Manos).

Others

1.CodeHi Control Upgrade to V1.0.2.2(DBCS Fonts-Chinese etc- Scroll Bug fixed and vertical scrollbar sometimes covered by the split).

2.Updated API Files.

3.Set Current directory to Project path before making.

4.If there is /OUT in Project options, check if there is path specified;If not add project path to it before executing compiled exe.

That's all folks!

Regards,
