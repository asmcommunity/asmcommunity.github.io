Object file converter by Agner Fog
Agner Fog released a very nice and usefull tool :

This utility can be used for converting object files between COFF/PE, OMF, ELF and Mach-O formats for all 32-bit and 64-bit x86 platforms. Can also be used as a cross-platform library manager, a dump utility, and a powerful disassembler. Source code included (GPL).

File name: objconv.zip, size: 511087, last modified:2007-Jun-07.


http://www.agner.org/optimize/objconv.zip
