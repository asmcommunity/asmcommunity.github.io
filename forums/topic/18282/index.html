HLA v1.64 is now Available
HLA v1.64 is now available from the HLA Downloads page (http://webster.cs.ucr.edu/AsmTools/HLA/dnld.html). There have been a few minor improvements to the assembler (e.g., fixed a bug in the SIMD mov instructions, like MOVAPS, which didn't allow two register operands).

There are several new routines added to the HLA Standard Library, and a problem with fileio.<hexinput> has been corrected.
