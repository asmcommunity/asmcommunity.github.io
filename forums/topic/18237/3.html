Notebook System for Software Developer
Newsflash: Catalysts are the only drivers for Radeon, and all replacement drivers such as DNA, Omega are based on it.

I use the regular Catalysts with a small patch (officially ATi doesn't support them, so the chipset is not listed in the installation file. Patch fixes that, so you can install them).

And my IGP340M pretends to be a Radeon with hardware T&L and vertex shaders. But the performance seems rather poor on heavy vertexload, and I can see no difference between hw and sw vp, so I assume it is emulated.

Its featureset is actually quite nice. It supports 3 textures per pass, cubemapping, and both embm and dot3. So it has all the ingredients for some serious graphics fun.
I have used it to develop D3D9 stuff on the road, and it coped very well actually.
