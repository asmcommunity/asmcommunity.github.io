Radasm Help plz
I've changed my code block from 

 

[CodeBlock]

1=$ proc,$ endp,,,6


to 
 

[CodeBlock]

1=$ proc,$ endp,,,2


The value 2 is a flag that tells RadAsm to show a divider long only when a proc is collapsed. 6 is the logical or result of 4 and 2. 2 is to show the divider line and 4 is to have the ending line collapsed also. I think you want to see the ending line so 2 is all you need.
