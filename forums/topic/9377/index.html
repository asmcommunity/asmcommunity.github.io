Posted on 2002-12-05 18:21:56 by comrade
For Privalov (about ASMEDIT control)
This is not a bug, it's because of AES_SECURESEL style, that is applied to ASMEDIT control when you have selected "Secure selection" option in the "Options" menu of FASMW. In the AsmPad example no special feature styles are applied, so it behaves more like the standard Windows editor.
About such selection option - it would need only changing selection displaying algorithms, because spaces after the end of line are not copied to clipboard anyway.
