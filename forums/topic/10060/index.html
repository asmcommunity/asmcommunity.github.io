Dialogs2
The dialog system has been updated to handle most of the common controls, a couple of more examples have been added and it now has a help file to make the system easier to use. Minimum file size in the example code is 1536 bytes so the system appears to be capable of low code overhead for dialog boxes.

The README.TXT file explains how to set the system up in MASM32.

I would appreciate any feedback on using this system.
