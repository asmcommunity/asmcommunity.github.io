Re: ShellExecute with an internal dos command?
you can start the DOS window and use the command as parameter, like this

ShellExecute (Handle, "open", "cmd.exe", "/k dir", 0, SW_SHOW);

have a look at the possible parameters here: http://www.microsoft.com/resources/documentation/windows/xp/all/proddocs/en-us/cmd.mspx?mfr=true
