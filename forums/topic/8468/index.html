Reposted batch files for MASM32
I could not find the last posting myself so I thought it better to repost the batch files for MASM32. All that has been done with them is to enclose the names in quotes so that the batch files can handle long file names and path and file names with spaces. QE and ML.EXE/Link.exe can all handle long file names and spaces.

Unzip the small zip file into a directory of its own then copy them over the existing ones in the BIN directory in MASM32. Then the file name problems should disappear.
