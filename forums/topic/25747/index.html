Integrated ASM Wiki Book
I made a mod for SMF that allows you to view the ASM Wiki Book through these forums. It is much faster and more secure than the previous "MediaWiki" software. The old "/wiki/" site and related software have been removed. The link on front page of ASM Community has been updated to reflect this change.

New Wiki Link
-----> http://www.asmcommunity.net/board/index.php?action=wiki <-----

The integrated version should look and function practically the same, the only major difference is that it is "themed" by the forum software. Whatever forum theme you are using is applied to the Wiki.

PLEASE NOTE

  • Only the Community Staff is allowed to edit pages for the time being, we will notify members of this community as to when you can submit information to the wiki.

  • I will be actively working on the Wiki mod, so please expect some downtime for maintenance.

  • If you see/find anything that isn't working or doesn't look right in the Integrated Wiki, please reply to this post about it.

  • Any other questions/comments/concerns??? Reply here, please :)
