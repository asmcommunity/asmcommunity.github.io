-1 + 2 -3 +4 -5 + 6 .......+ (-1 )^n N ?

=(-1+2) + (-3+4)... + [(-1^n)*N]
=1+1+...x

where x is either 1 (if N is even) or x is -N (if N is odd)

sub eax,ebx
dec eax ;balances off for fall-through
@@:
inc eax
