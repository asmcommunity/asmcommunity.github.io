mov al, szString[5] [color=green]; eax now equals 0[/color]
There is no conversion from an ascii character to a number because they are the same thing.

mov eax, 'q'

is the exact same thing as saying

mov eax, 113

is the exact same thing as saying

mov eax, 71h
