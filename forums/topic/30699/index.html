For coding/testing Win x86 apps in a Win x86 environment on an x64 host machine you have some options.
You can install VirtualBox or VMware to boot the x86 Windows OS.
You cannot use Microsoft's VM for booting since you only have the home edition ( you need pro or ultimate).
