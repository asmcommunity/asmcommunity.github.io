GoAsm for RadASM
I have updated the RadGoAsm file on my website, the old one was getting a bit out of date.

http://www.assembler.ca/files/RadGoAsm.zip

The readme.txt file explains where to place the files and what is expected in certain folders. The GoAsm.ini file for RadASM assumes that you have installed the SDK for Windows 2003 SP1 available here:

Windows 2003 SDK

The fpu.kwl file has been updated to include most of the FPU commands, the old one was missing quite a few. The GoAsm.kwl file has been updated to reflect the latest help file in the GoTools distribution.

Both the x86 and FPU RadHelp options expect x86eas as the main help file, it can be downloaded here:

x86eas.zip
