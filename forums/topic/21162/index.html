dll2inc tool for C run-time DLLs
Dll2inc is designed to create module definition & include files from C run-time DLLs such as msvcrt.dll and crtdll.dll
To avoid naming conflicts with the original MASM keywords such as div and fabs, a macro named cinvoke is designed to call C functions.
The creation of import libraries from C run-time DLLs is very easy, check the zip file to see how to do it with Pelle's powerfull librarian Polink.
