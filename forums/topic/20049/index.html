MasmEd version 1.0.1.1
Masm assembly IDE

Features:
- Masm code editor with syntax hiliting.
- Resource editor with more than 30 windows controls + custom controls.
- Single exe file. No ini files or dll's needed.
- No project file.
- Ideal for beginners as well as more experienced.

Exe only: http://radasm.visualassembler.com/Upload/MasmEd.zip
With sources:http://radasm.visualassembler.com/projects/MasmEd.zip
