telophase,

iblis is right here, you use EnableWindow() both to hide and to show the window(control) you have in mind.

You use the two flags TRUE or FALSE to indicate which you require. In you startup code in the WM_CREATE or WM_INITDIALOG you use EnableWindow() with the FALSE flag and when you want to show the window (control) use it again with the TRUE flag.

Note that you must have a valid window handle for the window or control.
