Can't find kernel32.inc
Icz's tutorials are based around the MASM32 package which is of Hutch's design (& maintainence). This provides many useful things to you the programmer that MASM on its own doesn't.

One such feature is a list of definitions which cannot be extracted from lib files (ie constants), these were all placed in the windows.inc file which you are missing.

You should be able to get the MASM32 package from www.movsd.com which is Hutch's site. If not, just search google for "MASM32" rather than plain old vanilla masm.
