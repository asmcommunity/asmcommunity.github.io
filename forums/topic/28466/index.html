AsmEdit - New Version
I have upgraded AsmEdit to V4.0. Again I would like to thank all of the testers. 

New stuff:
1. Added a Tree and List views.
2. Wrote a new standalone Setup program that will allow you to install
   the AsmEdit package in any directory with any name.
3. Added some new features. e.g. The Run button on the Toolbar will attempt
   to find the associated exec for the displayed file and execute it (It
   could be any file in a Project or Program folder, .asm, .rc, .inc, etc.).
4. Fixed some major bugs, and many minor bugs.
5. The program takes a second or two to load because it's expanding
   the Tree from the last used path. (I need to come up with a faster
   way to expand the Tree from a path).
Could use some idea's in this area. Also how could I access the users Unpack program from my Setup program and automate the install process. Or any other idea's on an automated install. Because right now the user still has to unpack the AsmEdit package to a directory. If you have the time check out the program.
