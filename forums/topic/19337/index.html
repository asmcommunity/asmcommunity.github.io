Windows Graphics Foundation API and Avalon
Old news?

There's an article on Slashdot called The end of DirectX as we know it that links to an article on ExtremeTech called Microsoft Longhorn Reinvents Desktop Graphics.

From Slashdot ...

"Speaking with ATI and NVIDIA at ECTS allowed us to confirm that after DX9.0, DirectX Graphics is no more. In name only. Microsoft's next set of core presentation and 3D APIs are now under the umbrella of Windows Graphics Foundation and Avalon. Microsoft will still rely on DirectX in name for the rest of the core components, but the graphics API is now under a new name. Look out for WGF 1.0 compatibility on the back of that next generation graphics card's box."
