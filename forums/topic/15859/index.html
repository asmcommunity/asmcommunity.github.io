An alternative linker for Masm/Fasm/GoAsm
Hi friends,

The COFF linker Polink from Pelles C package can be used to link Masm/Fasm/GoAsm object files. This linker has allmost the same command line parameters used by MS Link.Polink can use Masm import libraries;just it cannot handle OMF type object files.

Polink creates smaller executables compared to MS Link, Iczelion's paint example reduced from 3072 to 2048 bytes.

Pelles C software development kit is free to use:

http://www.smorgasbordet.com/pellesc/fulleng.zip

Homepage:

http://www.smorgasbordet.com/pellesc/

Thanks to Jeremy Gordon for his GoAsm example source code.
