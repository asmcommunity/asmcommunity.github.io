set dialog colors?
Because of the standard font, Windows uses after "CreateWindowEx".

Use this to change it to the "normal" font: 



        invoke GetStockObject, DEFAULT_GUI_FONT

        mov hFont, eax

invoke SendMessage, hButton, WM_SETFONT, hFont, 0
