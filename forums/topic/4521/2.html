hi stanks

In RadASM.ini you find this section:


;F1-Key=HelpFile
F1=$H\Win32.hlp
CF1=$H\x86eas.hlp
SF1=$H\Masm32.hlp
CSF1=$H\Rc.hlp

Change it to:


;F1-Key=HelpFile
F1=$H\Win32.hlp
CF1=$H\opcodes.hlp
SF1=$H\Masm32.hlp
CSF1=$H\Rc.hlp

The help menu has an option tool to help you set it up.
