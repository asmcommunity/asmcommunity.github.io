Check out source code of ProEmulator from SF CVS with ease
Extract the archive to a directory. Run checkoutProEmulator.bat to check out latest source code of ProEmulator (files are stored in the sub-directory 'ProEmulator'). Run checkoutARM7TDMI.bat to check out source code of ARM7TDMI plugin.

* Download the DLLs if u r prompted missing of them.
