New GameDev Tech
There are some new technologies that I will be backporting to ObjAsm in the near future.
Particularly, there have been major recent advancements in Artificial Intelligence which warrant immediate transliteration and optimizing. An example is Behavior Trees (see my blog on this site).

Further, there have been some recent advancements in object oriented paradigm which need to be considered and evaluated, such as "runtime mixins" (runtime constructed component based class objects), and "service component registration schemes" for more highlevel interfaces that require timely updates.

Lots of stuff happening!
