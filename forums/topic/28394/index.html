floating format
You can use the 3 IEEE formats. REAL4 => IEEE 32-bit real numbers REAL8 => IEEE 64-bit real numbers REAL10 => IEEE 80-bit real numbers Read the chapter 6 of the MASM manual for more information. Good luck
