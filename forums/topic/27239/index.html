smurf, http://www.pbq.com.au/home/hutch/tools.htm Download the file called NRE.EXE, its old but it does the job OK. Be careful and only use numeric IDs for controls and dialogs, its the recommended method by Microsoft anyway, it will do dialogs, icons and bitmaps OK, its worth learning to edit menus manually as its faster and easier to edit. the trick with it is to delete out the C++ junk and put the include to the resource.h file in the masm32 include directory.
