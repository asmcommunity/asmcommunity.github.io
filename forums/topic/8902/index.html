I use this sequence in C
==============
int ndx = GetWindowTextLength (hEditLog);
SendMessage (hEditLog, EM_SETSEL, (WPARAM)ndx, (LPARAM)ndx);
SendMessage (hEditLog, EM_REPLACESEL, 0, (LPARAM) ((LPSTR) szBuf));
==================
