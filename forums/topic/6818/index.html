Simple Morse Code
Hi All Friends!

I think convert Standart Morse Code to Hex. Morse Code items like binary numbers.

I accepted short item (.) = binary 0 and long item (-) = binary 1. For example, national standart
help code is s.o.s and morse code is : ...---... if us want convert this code : 000111000 ok.

Result is like this: 



 win32asm =  .--..-....--..---.-...-- (18) 6433A30000 = 011001000011001110100011


First number is always total bit lenght. I used bit length because last byte not complete mostly.
