Shawn,

The version of ML.EXE in MASM32 is the original one from the win 98 DDK which is 6.1d upgraded to 6.14 with the patch from Microsoft. The patch comes with a text file that explains all of the changes and later stuff it supports.

6.15 works OK and I think it adds another data type for working with SIMD but it does not seem to do anything else better as far as I can tell. If you get the VC PP with the link that Ricky posted, you can open it with Winrar and just extract the files you need if you don't want to upgrade you version of Visual C with it.
