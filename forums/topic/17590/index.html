Script -> comment line ?
I found very usefull option in qeditor for masm - comment line, which is called from Scritps menu, and it adds such line to project.

;?=?=??=?=?=?=?=?=?=?=?

You could go further and add Scripts/comment block option with editable comment templates:
i.e if you have text 'SOMETEXT' and apply such option it shoud format this block

;?=?=??=?=?=?=?=?=?=?=?
;?=
;?= SOMETEXT
;?=
;?=?=??=?=?=?=?=?=?=?=?

In MASM you can also just do this, RadASM will hilite it with a different color that you choose from the "Colors and Keywords" dialog: 

mov eax,10



[color=blue]comment /*



SOMETEXT



*/[/color]



add eax,50


If you select a block of text and then use "Comment" from the Format menu it will insert a ; before each line in the block.
