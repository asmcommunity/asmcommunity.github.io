Casting
Actually there *is* casting in masm - if you're using the high-level ".if" stuff and need to do signed compares. Take a look at the code output from: 


.if	eax < 0

	mov eax, 10

.endif

.if sdword ptr eax < 0

	mov	eax, 10

.endif


...and then of course there's "this register points to a structure" kind of typecasting..
