RS32.exe
Are you using RC.exe (included with MASM32 package) and
compiling your *.RC file ? If not then this is how I compile my
resource files...

RC /r /v myresource.rc

This will output the *.RES file that the linker is looking for.

Good Luck !

Edit -
Forgot to mention that you might need to 'include' resource.h
in your *.RC file.
example
#include "c:\masm32\include\resource.h"
