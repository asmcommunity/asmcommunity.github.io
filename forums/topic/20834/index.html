Re: Using MASM How can i make a own smtp server and send out a mail from the server?
SMTP is basically a PLAINTEXT protocol.
Connect to an smtp server (usually port 110) and send the greeting "HELO domainname".
Catch the reply and process it.
Now read the SMTP protocol RFC document for more information.

There's a couple of "email api" you can use to do it, but it's much more fun and educational (and you are cooler) to write this as socket code.

Your first goal then is to write a TCP socket application which can, if nothing else, attemp to connect to a specified remote host on a given port.
