HLA V2.10 is now up on Webster
Hi All,
HLA v2.10 is now available on Webster at the usual download address:
http://homepage.mac.com/randyhyde/webster.cs.ucr.edu/HighLevelAsm/dnld.html

The changes to the compiler are mainly internal. (Hopefully) you will not notice any differences in the compiler itself.

This release includes HOWL v1.2; lots of changes based on the tutorial I'm writing.
The HLA examples download contains rewritten HOWL examples. The old examples may break with HOWL v1.2, so download the new examples if you're playing around with HOWL (the HLA Object Windows Library).

Cheers,
Randy Hyde
