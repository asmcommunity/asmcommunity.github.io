Connecting to a PHP-based server. how?
The simple solution would be to forget PHP sockets and simply send the server a HTTP request in the form of

GET /highscore.php?mode=update&user=Test&score=1000

If that isn't secure enough, you could use authentication as well, though that would complicate the request a bit.

If you really want to have PHP handle this, you'd have to have a PHP file running permanently, which doesn't sound like a good idea to me. You'd have to turn off tons of timeouts in PHP and server configuration.
