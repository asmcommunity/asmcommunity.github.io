SplashScreen Library 1.0
Hi @ all...
I've made this very little and simple library, which may be usefull to anyone who wants to add a SplashScreen in his application in just 1 instruction. The zip file contains the lib, the inc, the source code and a txt which explain the usage.

Main features of the library are:

? only 1 instruction needed
? only 1 bitmap stored in the resources needed
? customizable display time
? different thread: SplashScreen lets the main application continue the initialization

I hope someone will like and use it :grin: .
