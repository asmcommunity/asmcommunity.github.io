The GetTickCount api is what your looking for 


          invoke GetTickCount

           mov     eax, ADDR buftime1

           ..........................

           ..........................

           ..........................

          invoke GetTickCount

          mov     eax,ADDR buftime2

          sub      ADDR buftime1, ADDR buftime2

this is how I would write I'm sure others might have a better way of writing it. Hope it helps
