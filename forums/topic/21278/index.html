If you don't want to read RFC and other documents then do I it as I did when making IRC Client, all you need is Client, Server and TCP/IP Builder programm.

1. Run TCP/IP Builder as server (listen button).
2. Connect with client, and you'll see what's client sended you.
3. Now, connect to server with TCP/IP Builder, and send the samething what you got from client.
4. And again run TCP/IP Builder as server and send to client, what server returned you.

Keep forwarding packet till you log in...
