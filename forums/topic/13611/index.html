Displaying different image formats
at first, you can only display BMP images, since these pictures contain uncompressed and "windows prepared" data.
you will shurely find an example for displaying bitmap pictures among the assembly tutorials.

for displaying TIFF and TGA pictures you will need a decompression / decoding routine. Just check out the
documentation of the appropriate file formats. In result you will get a decompressed bitmap for displaying like above.
