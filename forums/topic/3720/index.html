How to connect using proxy server?
Depends on what kind of proxy it is. If it is a transparent proxy then you don't do anything differently. If it is a SOCKS proxy then you need to do some research on it. The easiest thing for that would be to get hold of some libs and headers for SOCKSv4 or v5 and convert them to masm for use. I think Thomas wrote some stuff using SOCKS. If it is simply an http proxy(yuck) then specify the entire URL in every request to the server after you connect to the proxy server in the normal way.
