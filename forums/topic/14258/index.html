WinAsm V1.0.1.3
Hi All,

This is the new version of WinAsm (V1.0.1.3)

What's new:

1.You can now navigate your projects from the new "Procedures" Tab on the Project Explorer.(Thanks TBD)

2.Add File now allows Multiple Files selection (Add File... renamed to Add Files...)Thanks TBD & Manos.

3.Several Improvements for PopUp Lists (Intellisense).

4.A little bit of reorganisation for Tools-Options-General to accommodate "Open Last Project On StartUp" (Thanks TBD).

5.Procedure Combo now allows Keyboard selection.

6.Quiting application is smoother

7.Some flickering of children when Opening a project removed.

8.Default "Res To Obj" option in Make Options removed.

9."Execute" looks for FirstASMFileName.exe not for ProjectName.exe if /OUT not specified(Thanks h4ng4m3).

10.Transparent icons in Tools Manager (Thanks TBD - I didn't see a problem with document icon in menu near file).

11.Double click on the selection bar on a line that corresponds to a PROTO declaration and you will automatically taken to the corresponding procedure (Thanks maCo).

12.A new menu item in the Project menu, when active file is a RC file, to open it in an external resource editor(Thanks Masmer).

13.Minor changes to CodeHi.dll Control.

14.Several Improvements and Optimisations.

Enjoy,

akyprian
