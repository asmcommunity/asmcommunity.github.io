In case your compiler operates in C++ mode, 


extern "C" void MyAssemblyProc(int a, int b);


will allow you to call 



MyAssemblyProc PROC C, A:DWORD, B:DWORD

  ; yadda yadda

  ret

MyAssemblyProc ENDP
