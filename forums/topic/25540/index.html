AniGIF V1.0.4.0
Hi all,

AniGIF V1.0.4.0 has been released. AniGIf is a custom control that supports both simple and animated gifs. It supports several custom styles and messages. You can get it from the Custom Controls forum at http://www.winasm.net. Includes full source of the control and a test application.
