I just spent near an hour hacking away some code to create a structured file storage, only to get error 800300FE, which is defined in Winerror.h as follows: 



//

// MessageId: STG_E_UNIMPLEMENTEDFUNCTION

//

// MessageText:

//

//  That function is not implemented.

//

Moral: read the MSDN page all the way to the bottom before coding (where it says UNSUPPORTED in Windows).
