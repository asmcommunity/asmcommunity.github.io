supi\rama, You have basically 2 types of calling conventions involved, STDCALL and C. Windows API calls almost exclusively use STDCALL. To call the CPP functions from MASM, you probably need to specify the C calling convention in the prototype you write for it. YourFunction PROTO C :DWORD,:DWORD etc.... Regards, hutch@pbq.com.au
