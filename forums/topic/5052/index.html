.rad file
Hi cakmak

The .rad files are used by the DlgToWin addin. The addin places a menuitem on the tools menu, Dialog to Window, and are used to auto write code from a dialogbox. You can choose to let the addin write all the code for a running app or only parts of the code. The files are plain text and can be chaged to suit your programming style.
