Multiple Soundoutput at once?
Yes it is: use DirectSound ;) make a Primary sound buffer and set it to continouse play, then make several Secondary sound buffers, load wav files into eacho of them, using the Lock.... (like in DirectDraw)..then start to play each or every of them...Primary buffer will do the mixing for you ... or you can do the mixing and special FX using CPU and custom Algorithms...then send all data to the Primary buffer...your choice here ;)
