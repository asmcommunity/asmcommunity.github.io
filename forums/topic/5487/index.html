To Bighillwill,

Yes for the time being you will need a duplicate
copy of Hutch's Bin, Include, and Lib files in
the AsmEdit's root directory.

The reason it was originally set up that way was
because I had a lot of additional files in the
Bin and Include files that are now no longer
needed.

But still you won't be able to assemble AsmEdit
or some of the support program using Hutch's
Windows.inc, because of missing elements.

The next release of AsmEdit will still use the
duplicate files, but the following release will
not.

It will take some work and a lot of testing to
switch over.

Ewayne
