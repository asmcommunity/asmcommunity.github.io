There's a sample on how to load a bitmap on the masm32 package it's on DriveLetter:\masm32\EXAMPLE4\BITBLT it has everything you need to know on how to load a bitmap. As for resource files compile it using the resource compiler(rc.exe).

also study the assemble and linking parameters, this is where it'll make sense on how to combine the .obj file and .res file into .exe
