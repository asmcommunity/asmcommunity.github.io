Layered Window
This works only on 2000/XP

Put this into WM_INITDIALOG, makes the window 80% transparent. 


invoke GetWindowLongA,hWin,GWL_EXSTYLE 

or eax,80000h                                                     ; WS_EX_LAYERED = 80000h

invoke SetWindowLongA,hWin,GWL_EXSTYLE,eax

invoke SetLayeredWindowAttributes,hWin,0,(255*80/100),2           ; LWA_ALPHA = 2
