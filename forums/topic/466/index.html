Cinchy server
This worked for me:

1) download it from Iczelion's
2) rename "server.rc" to "rsrc.rc"
3) copy an icon into the directory and name it "s3.ico" since he didn't include his icon
4) in rsrc.rc, edit the line #include "resource.h" to have a full path, ex., #include "c:\masm32\include\resource.h"
5) build the resource file
6) build the project
