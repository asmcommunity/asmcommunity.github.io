AsmEdit V4.2 (Released 11-05-2001)
New features. 



1. Added a Addin menu and maintence routine that will

   allow you to add, insert, insert a seperator, add a

   submenu, edit, and remove an almost unlimited number

   of Project Items.

   The menu Items can point to .dll or .exe type programs.



   Once the Addin menu is setup you can create a Project

   via the Addin menu and it will create a folder and the

   necessary files of your choice to a target directory

   of your choice and add a menu item in the Addin menu.



NOTE!  Read the ReadmeMenu.txt for the Addin menu setup

       and maintenance instructions.



   The projects for the Addin menu need to be designed

   with an interface to AsmEdit, there are samples and

   templates available.



  A. Utility and ProjTest are samples.

  B. ProjTemp is the template, look at the template for the interface.

     (Click on the Projects>New Project DLL Addin Template Menu.)



2. Added a Auto Indent feature.

3. Included vkim's debug feature in AsmEdit, copy the 

   Debug.inc and Debug.lib files from >Programs>Debug

   to AsmEdits include and lib directories.

4. Included a DebugTest for the vkim debug and DSPTest

   for my DSPValue in the Project Tree.

   Just open their folder, display their source and press RUN.

   You can match the source commands with the displayed output.

5. Some additional features, plus some misc fixes.

_____________


If you have already installed the AsmEdit package you can unpack
V4.2 to the same directory, but if you have some of your own
Programs or Projects you may want to back them up just in case.
Your current AsmEdit registry entries will remain intact.

Run the Setup program even if the files have been
copied, so the paths will get updated.
