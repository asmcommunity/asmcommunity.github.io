RadASM: masm struc / struct
Hi all

To make RadASM handle both masm STRUC and STRUCT definitions, make the followin changes in masm.ini:

Add this line to section

xx=struc,{T},{C}{I}{$} ends

xx is next free number.

Change this line in section 




Struct=$-,ends,struct,struc
