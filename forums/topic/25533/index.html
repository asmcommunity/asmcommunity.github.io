AniGIF V1.0.3.0
Hi all,

AniGIF V1.0.3.0 has been released. AniGIf is a custom control that supports both simple and animated gif's. It supports several custom styles and messages. You can get it from the Custom Controls forum at http://www.winasm.net. Includes full source of the control and a test application.

What's new
New message supported:
-WAGM_SETBKCOLOR

-New menu item:Options-Background Color

-minor bugs fixed
-Accelerators and menu shortcuts added

Enjoy,
