Fresh 1.0.0A pre-alpha
Hi all.
This version does not introduce some new features, but only change the project structure. Now in the project are used some technics, that may be useful not only in the Fresh but for general programing too.

Quote from 'history.inc'
[27.09.2003] v1.0.0A - pre-alpha

- IMPORTANT: New procedure macroses. New code should be write this way.
- Basic windows include files added to the package.
- Entire project converted to use StdCallEx.inc
- New StrLib library calling convention implemented.
- Some small optimizations and bug fixes in strlib.
- Introduced procedure "JumpTo" for size optimized message handling.
Look at sources of some window procedure for example:
MainWindow.asm|MainWindowProc
[24.09.2003]

- Fixed bug in controls.asm about handling WM_CLOSE in the
form editor.

[16.09.2003]

- New about box added.
- New standard library: mainloop.asm It is standard main message
loop code and procedure ProcessMessages.
