MCI -> How to get the song list from cd?
You cant get the song names via MCI because they are not stored on the CD.

Easiest way would be to get the CDDB ID and make a CDDB query. That way you get the Album name, date, Artist and title & length of each song (plus some other info like Genre etc.)

IIRC, there is a SDK available on www.cddb.com which has some easy to understand example code :)
