Will,

Thanks for passing back this comment. The Ctrl+V etc ... are actually built into the rich text editor and while I trapped and changed a few others that are the same, I left the standard clipboard operations alone as many people like using them.

I personally use the older Ctrl + Insert / Delete type operations as they work on almost anything.

What I did change with the standard key combinations was the message it generated, if you leave WM_PASTE as normal, it can preserve the font, colour and formatting from another source so I use EM_PASTESPECIAL with the constant
CF_TEXT instead from all of the key combinations so that it is pasted as plain text without the extra formatting.

I don't know what is happening apart from that, it tested fine on 95b, 98se and win2k.
