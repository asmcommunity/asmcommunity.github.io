calling asm code from C/C++
You declare a prototype to your function in C. 
return_type __stdcall func_name (param_types params);
and in your asm file: 
func_name proc stdcall params:param_types
   (code)
func_name endp
and execute your C/C++ compiler to make: program.obj file and execute asm to make: asmprocs.obj file and link program.obj with asmprocs.obj to make: program.exe
