Re: Shareable section in C++
for Visual C++:

#pragma data_seg(".shared")     // Make a new section that we'll make shared
HHOOK g_hHook=0;                // HHOOK from SetWindowsHook
#pragma data_seg()

...then apply the correct linker switches to make the section sharable.
