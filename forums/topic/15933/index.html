TerminateProcess ?
Hi, martidim
Using TerminateProcess is a bad idea, you are supposed to kill apps like that if there is no other choice. Try closing the main window, sending WM_QUIT to it's threads, and if all fails then TerminateProcess. That way your program would trty to close apps gracefully first.
Cheers. :)
