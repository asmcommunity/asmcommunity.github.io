ProEmulator PluginWizard now ready for use
ProEmulator PluginWizard is a tool that generates source code skeleton of a ProEmulator plugin (currently only generates processor plugins in C language). By providing information and specification of the device to be emulated in the wizard, the skeleton of the plugin source code can be automatically generated.
