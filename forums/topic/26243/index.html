fat tables, direct disk access
To directly acess a logical drive or physical disk, you have to lock it using int 21h functions. To do this under Win32 in an application you have to use CreateFile to load the VWin32.VxD device driver. The using DeviceIoControl you can call most(if not all, including direct disk acess functions) Ring 0 disk acess interupts. For more documentation on this, seach the msdn developers library (At msdn.microsoft.com click on the library menu up the top, if search is there, use that (i cant remember) or click on Windows 2000 and then on Documentation in the Table of Contents menu that comes up, then search.
