Bit7,

you can use :-

FONT 8, "Tahoma",<weight>,<italic>

where weight is the weight of the font, 800 should be bold.

I think the italic goes in as a 1 or zero.

As for changing just one control, use a WM_SETFONT message in yout WM_INITDIALOG, to change the font of the dialog when it loads. I don't think there is a way to do it in the .rc file, but anyone please feel free to contradict me.

2) Can i draw a small icon (same size as in the window caption bar) inside the dialog ?


if you mean to display the icon somewhere in your dialog, then this should work: 


500 ICON "bla.ico"
]
in the dialog create a static control like this: 



Mydialog DIALOG DISCARDABLE  122, 112, 276, 130

STYLE DS_MODALFRAME | WS_POPUP | WS_VISIBLE | WS_CAPTION | WS_SYSMENU

CAPTION "Blaha"

FONT 8, "MS Sans Serif"

BEGIN

        CONTROL         500,IDC_STATIC,"Static",SS_ICON,5,6,267,62   //where 500 is the icon resource id

END


hope that helps..
