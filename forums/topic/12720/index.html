PMM 0.5b Released
I have released PMM (a heap dynamic memory manager written in FASM) version 0.5b today. PMM is a powerful memory manager supporting dynamic reference counting, reallocation, and garbage collection.

You can find PMM Version 0.5b on my projects page @ http://www.coggeshall.org/projects/
