A question about output path in RadASM
In RadAsm, can the output path of a project be set so that the exe or dll could be built in a certain directory?
Posted on 2004-03-30 02:00:27 by optimus
A question about output path in RadASM
Hi Optimus

If you are using the MS linker, then set link command:

5,OT,$B\LINK.EXE /SUBSYSTEM:WINDOWS /RELEASE /VERSION:4.0 /LIBPATH:"$L" /OUT:"$5",3,4

You can then use Project / Main Project Files to set the path and filename for the dll / exe
