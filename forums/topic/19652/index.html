Baby DLL for debugging
http://www3.telus.net/ldh/asm/debug.dll
It's MASM source, with make-file and def-file included, for a little DLL which dumps any small piece of memory to a message box. It's for "run-time debugging": you write a call to the dll within the source you are trying to debug. It's handy in some cases where a full debugger has problems.
