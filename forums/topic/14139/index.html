WinAsm V1.0.1.2 (Sixth Step To Intellisense And Other Goodie
Hi all,

WinAsm V1.0.1.2 is available for download from the WinAsm site.

What's new:

Intellisense

1.Type Include and you get a list with all asm & inc files of the project (only name+ext if in project directory, full path name + ext if in any other directory) plus all *.inc files in your Include directory. Type IncludeLib and you get a list with all *.lib files in your Lib directory.(I hope eko is satisfied with this solution)

Other

1.Major rework for reading *.vaa files---->much faster at startup.(Thanks TBD for reminding me although I've chosen a different approach.Is it OK now ?)

2.You can now define different colors for API Functions, API Structures and API Constants.

3.Copy selected text or all text of Output Window to the clipboard. Also all Out text can be saved to file "OutText.txt" in the project directory. You can access theese via a contextual menu (Thanks Masmer for similar request).

4.Option to show/not show Output window on successfull Make(Thanks maCo). Also Make progress is shown on the status bar .

5.In the Project menu a new item to Rename File (Thanks Jnrz & maCo - use with caution).

6.Upgrade of CodeHi control to V1.0.2.5

7.Errors in collapsed procedures now automatically expand procedures.

8.Caps Lock and Num Lock indications on taskbar altered a bit.

9.Various optimizations.

Bug Fixes

1.Bug Fix: When WinAsm goes to the notification the area, the context menu was not hidden if you decided not to choose anything form it (Thanks Jnrz).

2.Small Bug Fix: Ctrl+Space inserted a space. It Should not.(Thanks KetiIO)

3.Error bookmarks in collapsed procedures made the error line invisible. Only way to get it back was by reopening the project.(Thanks KetiIO)

4.Bookmarked lines in collapsed procedures were hidden after procedures expanded.

This is it,

Hope you enjoy this new release,
