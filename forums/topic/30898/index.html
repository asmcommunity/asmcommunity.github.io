Assembly written CMS.
I would like to announce one of my projects - MiniMagAsm - pure assembly written CMS web application. The web site on the above link is managed by MiniMagAsm.

It is portable between Windows and Linux and highly customizable - you can make the visual design of the site whatever you want, editing the template files, style sheets and images.

The source code is available in the fossil repository.

MiniMagAsm is not finished, but can be considered "beta" now.
