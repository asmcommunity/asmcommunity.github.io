Okay, I just recompiled all the code with HLA v1.47 and verified that I could successfully run the first 14 (or so) tutorials.

To avoid problems with HLA changing faster than the Iczelion translations in the future, I've moved them into the ICZ subdirectory of the "Examples" folder in the HLA distribution (i.e., it's no longer a separate download on Webster). I have a short testing script in my makefile for HLA that automatically makes everything in the Examples subdirectory. I usually run this before releasing a new version just as a sanity check to see if I've broken anything. Now that the Iczelion tutorial translations are part of this code base, I'll immediately know if I break them in the future.

In any case, you can get the tutorials by going to the HLA download page and download the "Examples" package.

Note that the code in the HLA v1.47 Examples directory compiles fine with HLA v1.47, but
I won't guarantee this with earlier versions of HLA (HLA v1.43 is the current *stable* version, v1.44-v1.47 are all experimental versions).

Cheers,
Randy Hyde
