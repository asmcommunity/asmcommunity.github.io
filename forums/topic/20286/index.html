1) Latest appy - FrameRectShow, specific window spy:
http://www.geocities.com/xmemor/2lz/framerectshow.zip (6kb)

Shows chromatic frame of every window within screen:

if edge of cursor is outside window => frame is black
if it within not-selected window => frame is white
if it within selected window => frame is yellow
(selects window with help of mouse)

shows some parameters of selected window:

alpha/-? visible/hidden? enable/disable? region_type? hWnd?
CtrlID? layer? processID? threadID? class? text? exefile?

plus something else with help of keyboard:

F8 => refresh screen
F9 => capture selected window
F10 => some operations for captured window (here - set window region to NULL)
+ release captured window
F12 => exit
CAPS LOCK "off" & SCROLL LOCK "off" & NUM LOCK "off" => shows all frames
CAPS LOCK "on" => fix selected process (PID), shows it's frames only
SCROLL LOCK "on" => fix selected thread (TID), shows it's frames only
NUM LOCK "on" => fix window's layer (N), selects corresponding frames only

if NUM LOCK "on":

DOWN ARRAY => N +1
UP ARRAY => N -1

if WinOS>=Win2k & selected window has WS_EX_LAYERED, i.e. alpha-channel:

HOME => set alpha-channel to full opacity (alpha =255)
END => set alpha-channel to full transparency (alpha =0)
PAGE DOWN => alpha -1
PAGE UP => alpha +1

Tested on win2k-pro-sp4, but must works on win9x/XP also, i hope...
