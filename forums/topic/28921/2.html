Re: Virtual Desktop 1.0.0 - Feedback request
Alright. A new update. Below is the list of the features added to this version:


      [*] You can now press the Enter key on any of the short-cuts in any of the Virtual Desktops to execute that item with its default associated program.
      [*] The resizing aspect of the program and the virtual desktops is really enhanced.
      [*] The position of the desktops and the tab control is changed depending on where the window is sticking to (right or left side of the screen).
      [*] The close button is added but the functionality is not working yet.
      [*] You can now rename your Virtual Desktops either by double clicking on their tab or pressing SHIFT+CTRL+ALT+R.
      [*] Once an item is deleted in your real HDD, its shortcut will also be removed from Virtual Desktop.
      [*] The contents of the tabs are now saved and restored every time you run the program. YEAY.


Shortcuts up to this point:


      [*] SHIFT+CTRL+ALT+D = Show or hide the program.
      [*] SHIFT+CTRL+ALT+R = Rename the current virtual desktop.
      [*] SHIFT+CTRL+ALT+W = Attempt to close the program.
      [*] SHIFT+CTRL+ALT+A = Add a new virtual desktop.
      [*] CTRL+TAB = Navigate between virtual desktops.
      [*] ENTER = Execute the current shortcut in the current virtual desktop.
      [*] ESC = Minimize the program into the tray. Same as SHIFT+CTRL+ALT+D while the program is showing.


Known issues:


      [*] The close button is sometimes not correctly painted.
      [*] Once you add many virtual desktops, the width of new tabs are calculated incorrectly and the scroll-bar will not be fully shown (if any).


That's it for now. I would really appreciate comments on this version. Any buys? Any features you want which you are not seeing? Please just let me know.
