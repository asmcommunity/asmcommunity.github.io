RadASM project properties made avilable to addins.
Hi all

New upload at the top.

- Made several minor improvements.

Most noticeable is the changes made to properties listboxes. Since the listbox can be given focus you can use the listbox inbuildt seach routine.
In dialog edit the listbox will hold its position (based on item name) when a new dialog control is selected. This helps a lot when setting up names, ID's or tabindexes for several controls. Also tab and enter has been given meaningful functions while the listbox or dialog has keyboard focus.
