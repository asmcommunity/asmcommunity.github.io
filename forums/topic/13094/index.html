Visual ASM (V1.0.0.6)
Here is the new release which is mostly bug fixes and minor improvements. Copy the files you will unzip (3 files) on top of the files of the previous release.

What's new
1.API Functions AND API Structures are now considered as keywords and thus coloured (All case sensitive except those few starting with a small letter).

Bug Fixes
1.Find (match case) bug fixed, Replace improved, Replace All bug causing Visual ASM to hang fixed. Thanks Masmer.

2.Bug Fixed:When procedures are collapsed, selection from the procedure combo did not scroll source code correctly. Thanks frankies.

Improvements
1.Procedure Combo flickering minimized.

2.Focus of main window, child windows and dialogs is improved.

3.Improved support for Function Tooltip position.

Regards,
