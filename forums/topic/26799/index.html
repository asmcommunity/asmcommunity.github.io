Masm32 and Soft Ice
For win95/98 SoftIce v3.24 is the only version I have been able to get to work. I also use a batch file to invoke the symbol loader. Contents of 'Debug.bat' nmsym /translate:source,package,always /source:. /load:execute,break %1 If you have a program to debug called MyProg.exe then type 'debug MyProg.exe' in a dos window. I don't know about Win2000.
