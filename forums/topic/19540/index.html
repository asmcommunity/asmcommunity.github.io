Adding Text Into Edit Box
1. EM_SETSEL: Set Selected Range.
2. EM_REPLACESEL: Replace it.

invoke SendMessage, hEdit, EM_SETSEL, iStart, iEnd ; i > Max is last.
invoke SendMessage, hEdit, EM_REPLACESEL, 0, addr zStr
