Step by step on how to create your own operating system
first you have to find on the internet like i did, a bootloader that loads your compiled code.

http://board.flatassembler.net/topic.php?t=12668

this will work for a usb

now you have to decide what to do, make a console application, a graphical interface or just a kernal

i strongly suggested these following steps

make a program called kernel.bin
make a picture show on the screen in graphical mode
set up a mouse
set up first icon
make icon run file system that recognizes hardware
make a picture editor
make a second kernel
make support high level
make a 3d animation tool
always have the system boot to first kernel so its fast
make system calls to kernel1 so that a window can be drawn or to enter full screen mode\

make a developer operating system thats simple to use comes with instructions to make system calls. one system call could be a kernel override which would give the user full acceess to the computer
