WinAsm V1.0.1.6
Hi All,

This is WinAsm Version V1.0.1.6

What's new:

1.OwnerDrawn menus.

2."Debug" menu item in Make menu launches external debugger of choice (Tools-Options-Miscellaneous-External Debugger) if current project is EXE (Thanks Masmer, andrew_k).

3.Find and Replace buttons on the toolbar(Thanks andrew_k).

4.Executing EXE's (incl. consoles) after "Go All" is now optional. Tools-Options-Miscellaneous-Launch EXE on Go All(Thanks andrew_k).

5.Include and IncludeLib path can now contain spaces. e.g. C:\Program Files\...(Thanks andrew_k).

6.File not saved indication removed. Save on toolbar and menu item grayed to indicate that file is saved(Thanks Manos).

7.Some minor behaviour change for Add-Ins.

8.Updated API Files.

Bug fixes:

1.Intellisense:Small bug fix for Structure members.

2.Small bug fix: When all bookmarks were cleared the toolbar was not updated until the left button of the mouse was clicked in the active document (Thanks andrew_k).

Regards,

akyprian
