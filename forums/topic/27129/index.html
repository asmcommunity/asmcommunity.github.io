API Funct for protecting memory
What do you mean by memory "owned" by a thread? Threads, generally, share memory with all other threads in the same process (running instance or copy of a program). They are protected from threads in other processes (Those other threads need to use a stealth function like ReadProcessMemory or WriteProcessMemory.) There is such a thing as Thread Local Storage, but I haven't looked into what kind of protection, if any, it provides.
