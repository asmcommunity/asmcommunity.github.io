Add New Procedure: Addin for Winasm
Hi All,
I've uploaded an addin for Winasm called AddProc. It is a simple dialog based addin that generates the code for a new procedure based on the details you enter or select. It has 7 'pages', each with a specific purpose. However, as it says on the
first page, you can click 'Finish' at any stage to enter the generated code. The readme has all the details.

A HUGE thanks to akyprian for the endless support, help and great ideas.

Note: This now contains an updated version with a couple of bug fixes
plus a right click menu option.
