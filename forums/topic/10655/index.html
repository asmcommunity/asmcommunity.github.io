Hi dude,

Glad to see you back. What you are after is simple, create the DLL in MASM, this will give you the library with the exported functions, write the prototypes for the functions in VC and include the library when you do the build. Make sure the calling conventions are correct between the DLL and the C prototypes.

Just make sure you use the appropriate EXTERN C syntax so you don't get mangled names.
