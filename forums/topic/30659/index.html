Hi lordgus,

This is a million miles from my experience, so this is a very wild guess!

I have seen problems with anything in "Program Files" - the space in the directory name confuses certain tools. If you are entering "Program Files" anywhere, make sure it's enclosed in quote marks. Even then, the quote marks aren't always passed from one tool to another. Using "progra~1" - with the "(x86)" in there, perhaps it's "progra~2" - might help. If you're not entering "Program Files" yourself, this won't help. I can't imagine why this would work once but not twice, anyway, so may not be your problem...
