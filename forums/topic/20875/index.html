Management Funny
Today at work a fellow co worker had a short conversation with the Program Manager of one of the projects we are responsible for ...

Manager: What is Foo written in?
Co-Worker: It is written in C for Linux.
Manager: Oh, really? It looks like it was written in Word, or some kind of Notepad thing.
Co-Worker: Ummm, that is just what you used to view the source code, any text editor will work.
Manager: Oh, um, ok.

We have been having a lot of fun with that today.  Just thought that I would share that with you all.
