RadASM 1.0.7 Released (01-04-2002)
RadASM V 1.0.7
-------------------------------------------------------------------

Warning: If you are updateing, make a backup.

Unzip RadASM.zip to C:\RadASM

Fixed bugs and improvements
-------------------------------------------------------------------
Fixed Bugs:
-Fixed some minor problems.

New Options:
-------------------------------------------------------------------
-Make, Help and Tools menu accelerators.
-More pre defined Menu editor accelerators.

New Features:
-------------------------------------------------------------------
-Right click menus on Project and Output tool windows.
-Lock / Unlock file (Read only).
-Improved Code Properties search.
Still needs to be manually refreshed if new code properties are added
to your code, but no need to refresh if lines are added or deleted.
-Flexible splash screen. Edit RadASM.ini to set bitmap, time it shows and
text to be displayed.
-Find now centers line vertically.

;F1-Key=HelpFile
F1=$H\Win32.hlp
CF1=$H\x86eas.hlp
SF1=$H\Masm32.hlp
CSF1=$H\Rc.hlp

This defines the F1 help accelerators. And you must change the ini to re-arrange them. You can not use any accelerators that is alredy used elsewhere in RadASM on the make, tools and help menus.
