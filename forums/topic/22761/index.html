Hi

It is really easy if you know what to do.
I dont have msdn 2005 so I cant give you the exact stepp by step.

Here is how you can install MSDN Library for .net 2003

1. Download and install the H2Viewer.exe to C:\RadASM\Addins
2. Use regedit to find the key you need. Search for ms-help:// until you find a key that looks like ms-help://MS.MSDNQTR.2003FEB.1033
3. Use this key to configure RadHelp

Yess, i did it last night.......great solution (thanks for KetilO) now i can drop good old Win32.hlp , apiguide ,  and delete msdn from my mozilla favourites  :)
The steps i did:
1.) get the H2Viewer.exe from here: http://helpware.net/mshelp2/h2viewer.htm and copy to Radasm\Addins folder
2.) get the latest (2006 mach) sdk from here: http://www.microsoft.com/downloads/info.aspx?na=46&p=3&SrcDisplayLang=en&SrcCategoryId=&SrcFamilyId=0BAF2B35-C656-4969-ACE8-E4C0C0716ADB&u=http%3a%2f%2fdownload.microsoft.com%2fdownload%2ff%2fa%2fd%2ffad9efde-8627-4e7a-8812-c351ba099151%2fPSDK-x86.exe&oRef=http%3a%2f%2fwww.microsoft.com%2fdownloads%2fdetails.aspx%3fFamilyID%3dd8eecd75-1fc4-49e5-bc66-9da2b03d9b92%26DisplayLang%3den
this is a web install, to offline pc you can dload the full 400mb install iso image from here: http://www.microsoft.com/downloads/details.aspx?FamilyId=E15438AC-60BE-41BD-AA14-7F1E0F19CA0D&displaylang=en
3.a) online pc: just start webinstall and select components (if you need only core this will be approx 700mb)
3.b) offline pc: burn iso, and run, or you can open iso content directly with totalcommander, and run
4.) run h2viewer, you can see the namespace of this sdk:  MS.PSDKSVR2003R2.1033
5.) make sure radhelp is enabled, open option/radhelp options
6.) select masm.ini to change its settings
7.) select winapi.kwl (to click the text) and overwrite the old $H\win32.hlp to this: ms-help://MS.PSDKSVR2003R2.1033 , i repeat this to the default and put checkmark.

This is far the best help system i've ever seen !        Good luck.....s5vi

Dont know if this has happened to anyone, but I had problems when running multiple times the H2Viewer from RadASM.

A dialog box appear saying that the application crashed but when looking with taskman, it remained in memory. An instance for each time F1 was pressed.
