Desktop Shortcuts
FindWindow for Progman should return a value something like 65638, under no circumstances should it return NULL except if Progman is not running. In that case it will return 0 and the FindWindowEx function will look for children of the desktop as a NULL handle indicates the desktop:

FindWindowEx
If hwndParent is NULL, the function uses the desktop window as the parent window. The function searches among windows that are child windows of the desktop.
