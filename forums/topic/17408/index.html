Creating com files using masm615?
First, you need to get the 16-bit linker to build COM files:

http://spiff.tripnet.se/~iczelion/files/Lnk563.exe

Example code: 


.model tiny

.code

org 100h

start:

	push cs

	pop ds

	lea dx,msg

	mov ah,9

	int 21h

	int 20h

	msg db 'Hello amigo!$'

end start


Building the COM file: 


ml /c Sample.asm

link /tiny Sample.obj ; 16-bit linker
