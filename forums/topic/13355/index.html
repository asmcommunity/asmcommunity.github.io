Afternoon, dion.

They were requesting the serial number of the device to make sure you had a legit/legal machine.

All hardware have serial numbers. Just have a little look (maybe underneath, or on a pcb).

If the serial number has been removed, then this machine isn't legal and it would go against board rules to supply you with any info regarding it.
i.e. might have picked this machine up at a 'puter market, and plan on using it to print illegal tickets.


On the other hand, the serial number might have been a sticker which has come off and got lost at some point.

In that case, google for and grab the PDF file given in the seventh link given ( www.lai.com.au/support/manuals/ ripper_ribbit_manual.pdf ). It's for an arcade game which uses the ticket dispensoer, and has info on the dispensers electronics (power/input/output/etc).
