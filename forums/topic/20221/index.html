WinAsm Studio V4.0.1.258 (Intellisense is going intelligent)
Hi all,

WinAsm Studio V4.0.1.258 has been released. Among many other new features, any user function-trigger word is now allowed (not only "Invoke"). You can set all your triggering words from Tools-Options-Intellisense. According to your settings, unlimited number of function trigger words on a single line can exist with proper tooltip! More info and download in the WinAsm Studio forum.
