BareBones DLL - a simple demo
In response to some queries about making a DLL under MASM, I am publishing this BareBones DLL (by me).

It is comprised of a BatchFile used to build the DLL,
a DEF file which lists the Exported Functions of the DLL (one example provided)
and the ASM sourcecode for the DLL skeleton and example Export.

Hope this comes in handy :)
