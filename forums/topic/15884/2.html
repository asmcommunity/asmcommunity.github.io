For the full-text search to work, the section of your project
(.hhp) file must contain the following line:

Full-text search=Yes

If you're using HTML Help Workshop, you can add this line as follows:

1) Click the Change Project Options button, which is the topmost button on
the Project tab.

2) Click the Compiler tab.

3) Select the Compile Full-text Search Information checkbox.

If the full-text search facility fails to work after you have added the
line, it may be that one of the HTML Help runtime components isn't properly
registered on your system. When this is the case, you should get the
following error message when you compile:

HHC6003: Error: The file Itircl.dll has not been registered correctly

You should be able to fix this by following the instructions at
http://www.mvps.org/htmlhelpcenter/itircl.htm, or by using a tool such as
MJ's Help Diagnostics or HTML Help DLL Registrar.

MJ's Help Diagnostics: http://helpware.net/downloads/index.htm
HTML Help DLL Registrar: http://www.helpfulsolutions.com
