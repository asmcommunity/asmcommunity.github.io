Ok, the issue of flickering is pretty solved by the FAQ approach.

Thought the text on buttons still flickering a bit (and the one of the FAQ reply confirmed that). To get rid of it completely one needs to mend the fine ESF code example by changing the wc.style of WNDCLASS structure like this 



	;==================================================

	; Fill WNDCLASS structure with required variables

	;==================================================

	;

	; REPLACING

	;

 	; MOV	wc.style,CS_HREDRAW OR CS_VREDRAW \

	;		OR CS_BYTEALIGNWINDOW



	MOV	wc.style, CS_BYTEALIGNWINDOW



hope this helps :)
