If any of the other IDEs need a subforum
It would be great to get an EasyCode forum!

EasyCode is a MS Virtual type IDE for masm32. Currently there are no community groups for this IDE and it would be good to have a central location to report bugs and ideas.

The IDE has two modes, one where you virtually build the program with toolbar objects to form the majority of the program. Later you add the extra code to finish the application off. The other mode is pure masm32 code constructs with some added bonuses of module handling.

Thanks, Hiroshimator.

EasyCode's maker is a bloke called Ramon and his homepage is http://www.elprogramari.com/English/Assembler/AsmEng.htm
