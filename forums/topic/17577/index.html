feature request for RadASM
hello everyone!
first of all i would like to express my compliments to KetilO for the RadASM, after i downloaded it i re-discovered my old passion for ASM, thank you!

second, i would "dare" to make a request for a feature i would like to see in a future release.

i like and use very much the templates of RadASM (i think is one of the most RAD tools) and working with it i soon started to create my own templates for my own use and i have now a lot of different templates.
the problem now is that the window for the selection of the
templates is very small (just 4 lines) and the management, if we have a lot of templates, is not that easy.
what i would like to have is:
1) possibility to group the templates in sub-folders for easier management.
2) a bigger window for the selection of the templates, my suggestion would be to have in the "new project" wizard a new page after the first where you can select the template (or an empty project).
i would like to see something like the "professional" visual IDEs make when you create a new project (visual studio, delphi, etc).

i hope you like my idea and i hope it will help to further improve an already fantastic tool. (and of course i hope it will make the life easier to other developers like me)
