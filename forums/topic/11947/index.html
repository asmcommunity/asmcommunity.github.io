RadAsm HLA suggestion
Ketilo,

For HLA there are currently two project options console and dialog. I think there should be a win32 project option. I thought I could use the dialog option for any win32 program but I found out that if I use that option RadAsm expects a resource file. What do you think?
Posted on 2003-03-29 14:43:47 by Odyssey
RadAsm HLA suggestion
Hi Odessey

No problem.

In hla.ini

Change this:

Type=Console App,Dialog App

to:

Type=Console App,Dialog App,Win App


Add this:

Files=1,0,0,0,0
Folders=1,0,0
MenuMake=0,0,0,1,0,1,0,0
;x=FileToDelete/CheckExistsOnExit,
;(O)utput/(C)onsole/0,Command,
;MakeFile1[,MakeFile2[,MakeFile3...]]
1=
2=5,C,$R\HlaBuild.Bat -v -w,11
3=
4=0,0,,5
5=
6=
11=
12=5,C,$R\HlaBuild.Bat -v -w,11
13=
14=0,0,,5
15=
16=
