wininet.lib & speed ?
Get size of downloaded file. Remember it. Setup a timer for 1 second delay.
Upon timer fire up get size of the downloaded file again. Calculate the difference between this one and last one. This difference will be your bytes per second.
Than you can play with bps to make it kbps, etc.

Don't forget to update previous filesize to be equal to current one.
