cute file-problem...
No!! You cannot copy a file while there is a current file handle referencing it. Your options would be:

- copy it to a temp location, then open and use the temp copy for your work
- read the entire file into memory, then release the file handle so it can be copied
- copy it before you open it, or after you finish working with it

that's just the way things work in the Windows world ;)
