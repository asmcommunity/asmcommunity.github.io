tigru152,

The way to use the library in MASM32 is to include it at the top of your file with the line,.

include \masm32\include\masm32.inc

and following the includelib call to the library.

includelib \masm32\lib\masm32.lib

Once you have both of these lines in your source file, you can use any procedure in the MASM32 library.

There is a help file in MASM32 that documents the procedures in the library so it should not be a problem for you to use them.

Just make sure you spell all of the paths correctly as well.

I believe that you are trying to create a library from the looks of what you have posted!

normally you would use Lib to take all of your .obj files and then use a syntax something on this order to create the library.

Lib NewLib.lib -+ first.obj second.obj third.obj

and then add as many obj files as you like to the new library.

the PLUS and MINUS signes just tell the library manager to delete or add the obj's that are already included in a library if they exist!
