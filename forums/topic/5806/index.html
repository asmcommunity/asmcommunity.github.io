I had the same problem, put these commands in a batch file called buildit.bat and run it, or manually type them from the command prompt in the folder your source is in. It should then compile correctly.

ml -coff -c -Cx -DMASM6 -DBLD_COFF -DIS_32 message.asm
link -vxd -def:message.def message.obj

Also, Even when this works, you will Get more warnings than you can shake a stick at. Icz or someone really ought to clean that tute up a little.
