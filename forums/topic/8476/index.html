structure with arbitrary-size members
Try something like the following;
struc   RECT

{       .left           dd ?

        .top            dd ?

        .right          dd ?

        .bottom         dd ?   

        .size = $    }



struc   RGNDATAHEADER

{       .dwSize         dd ?

        .iType          dd ?

        .nCount         dd ?

        .nRgnSize       dd ?

        .rcBound        RECT    

        .size = $    }



struc   RGNDATA

{       .rdh            RGNDATAHEADER

        .Buffer        = $    }



virtual at 0

	RECT RECT

end virtual



virtual at 0

	RGNDATAHEADER RGNDATAHEADER

end virtual



virtual at 0

	RGNDATA RGNDATA

end virtual

Then to access lets say the 6th RECT in the RGNDATA struc use 
;eax points to a RGNDATA struc

lea eax,[eax+RGNDATA.Buffer+RECT.size*5]

;eax points to the 6th RECT

See this thread for more info.
