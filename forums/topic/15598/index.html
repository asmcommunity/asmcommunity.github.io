Hi bgong68,

This is the WinAsm "New Project" details:

WinAsm will not create a new folder for you.

If you choose to create an empty project, you simply choose its type (exe, dll etc) press OK and nothing else. There are NO changes to your PC yet. IF you decide to save your new project, you simply create a new folder (If you want so) from the "Save Project As" dialog.

If your new project is based on a template, you will be asked to choose the folder for your new project. Depending on your OS (e.g. WinXP), the "Browse for Path" dialog has a "Make New Folder" button, otherwise you have to use an existing folder.

I hope the answer is satisfactory.
