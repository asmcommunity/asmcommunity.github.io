HLA v1.44 is now available
HLA v1.44 is a quick maintenance release to support the RadASM and UeMake IDEs.
HLA v1.44 consists only of the HLA.EXE and HLAPARSE.EXE files, you will need to download a copy of HLA v1.43 first and then replace these two files with the v1.44 download.

HLA.EXE, v1.44, includes a new command line parameter that lets the IDE specify the location of the hla include files (thus avoiding the problems associated with the HLAINC environment variable).

HLAPARSE.EXE v1.44 has several fixes dealing with the temp directory option (the -p command line option).

Cheers,
