windows api references always to bit-masks, so you must use
<test>: 


	call	xyz_KeyFunction

	test	eax, FLAG_xxx

	jne	FlagIsSet



	; flag is not set



FlagIsSet:

	; flag is set



<test> works like <and>, but leaves the operands untouched.
