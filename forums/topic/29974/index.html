Seriously though, if you can't delete it for some reason, try this utility:
http://www.purgeie.com/delinv/

There's a 'bug' in Explorer's delete in that it works on a filename basis. With 'complex' filenames like these, sometimes something goes wrong somewhere in the parsing stage, so although the file exists, it can't find it when deleting.
This utility looks up the file ID directly, and then deletes based on that ID.
