Re: Disabling Buttons
Disable a button:
invoke EnableWindow, hButton, FALSE

Enable a button:
invoke EnableWindow, hButton, TRUE


To learn another way of doing that:
You can check the button's message documentation.
Or you can follow an important rule: to use the forum's search engine before asking.
