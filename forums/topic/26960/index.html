Tree View Control incorporated into a dialog box?
Try putting it into the dialog child control area like this: CONTROL "", RESOURCE_NUMBER, "SysTreeView32", X-pos, Y-pos, width, height, Style, ExStyle "SysTreeView32" is the registered class name. You can replace this with *any* registered class, even those you registered yourself (which is how MS itself can extend the 'standard' controls).
