what's about *.lpf?
Each file is 6400 bytes, representing an 80 by 80 square. Each value is either 'off' when 00h, or 'on' when 01h. Please, search the web for more info on the "game of life" to learn about cellular automata. The file just stores the grid in the program - red squares are 01h and while squares are 00h.
