BestWay for adding items to ListView
There is no optimized way to write to a listview, you must insert a new item and write each column. No way around it. The listview does not supply the capacity to add a group of items. The only thing I can suggest is that the code to add an item be made general and put in a proc so that it takes less space. Considering that it involves a minimum of 4 api calls there isn't alot of point to optimizing it anyway. The actual loading of the LV_ITEM structure is only an extremely small percentage (~<1%) of the execution time.
