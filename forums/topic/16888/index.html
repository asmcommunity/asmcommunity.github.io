I came upon an app like that recently and was/am quite interested in how it was done, it seems like a wonderful blend of treeview functionality within a listview. I ran a message spy on it and it seems to make clever use of regular listview messages such as LVM_INSERTITEM and image lists to create a "treeview" effect within the 1st column. I'm guessing it may use the iIndent flag of the LV_ITEM structure to indent the child items, but I'm not quite sure how the "branch" graphics are drawn.

The app is a freeware port monitor / process info tool called Vision 1.0, some might be more willing to analyze it rather than installing Kazza,
