HLA v1.67 is now available
Hi All,

HLA v1.67 (Windows and Linux) is now available on Webster at
http://webster.cs.ucr.edu/AsmTools/HLA/dnld.html

This update includes a few minor changes to HLA.EXE to support different suffixes (as well as a new command-line parameter for Windows users that lets you specify an executable name without a ".exe" suffix). It also includes an updated Linux version. This release contains minor clean-ups needed for the AoA CD-ROM master.
