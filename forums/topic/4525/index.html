Try linking with dxguid.lib :) its a full of GUID's library or use this:

; static guids for expected devices

;==========================================

GUID_SysMouse		dd	06F1D2B60h

			dw	0D5A0h

			dw	011CFh

			db	0BFh

			db	0C7h

			db	044h

			db	045h

			db	053h
