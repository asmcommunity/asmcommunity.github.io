Re: wierd functions in debugger
Usually you should debug while there is a PDB symbol database.
When I want to inspect the release-build, I put an "int 3" like this:


void BenchBreakpoint(){
	__asm{
		int 3;
	}
}


...
void main(){
	...
	BenchBreakpoint();
	..
}
