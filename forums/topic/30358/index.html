Re: searching simple to use 3D Scene-Editor
Well, I cannot offer you a complete solution, but perhaps you may find it useful to look at my BHM file format project. See also this thread: http://www.asmcommunity.net/board/index.php?topic=29830.0
It's a small and compact file format which I developed mainly for storing geometry and animation. The project on SourceForge contains a sample animation in BHM, a small OpenGL application which renders the animation, and the sourcecode for an exporter plugin for 3dsmax6.
It should not be that difficult to write a basic exporter for other modelers, or to write a converter for other file formats. Speaking of which, it would probably be useful to have a converter from Collada to BHM.
