BM_SETSTATE
I would like to have a button to be pushed in under program control as if the mouse were holding it down.

I have tried 

invoke SendMessage, !b1,BM_SETSTATE,1,0


to no avail.
Posted on 2005-09-24 00:05:45 by mikes
Re: BM_SETSTATE
invoke GetDlgItem,hDlg,05  ;05 is button id returns handle to button
invoke SendMessage,eax,BM_SETSTATE,TRUE,0


This makes button looked pressed for me.  Check to be sure !b1 is really the button's handle.
