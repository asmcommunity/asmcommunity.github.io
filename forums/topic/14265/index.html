HLA v1.53 Is Now Available
HLA v1.53 is now up on Webster:
http://webster.cs.ucr.edu/
and
http://webster.cs.ucr.edu/Page_hla/0_hla_dnld.html

HLA v1.53 includes several corrections to defects involving
code generation in FASM and Gas. Also, I've corrected the
emission of the ExitProcess call in respose to the EXIT statement
under Windows.

Also, I've modified the "Windows Programming in Assembly"
examples (in the "Examples" download package) to allow RadASM
projects using nmake.exe and batch files in additional to Borland's
make.exe program.

Cheers,
Randy Hyde
