Solar OS new version released on 2005_01_25
A new version was released.

Source code + binary is here:
http://www.hostileencounter.com/os/files/sol_os_src_2005_01_25.zip

Binary only here:
http://www.hostileencounter.com/os/files/sol_os_bin_2005_01_25.zip

New things in this version:
=====================
-Multiple Network drivers and Network cards can run in the same time.
-IRQ Dispatcher for PCI Steering implemented, devices can share IRQs now.
-Preemptive watchdog on IRQ-0 by cyberkilla04uk.
-Text viewer speedup on large/huge files.
-Removed some critical bugs: Malloc memory init.
-Other minor changes: MessageBox handling and Icons text clipping.

Aparently the first version that works in Quemu also

What is the PC's configuration?
What kind of video board and what kind of mouse.
You should use a PS/2 mouse since USB mouses are not yet supported. Serial mouse should also work.

In GUI mode:
You can reinitialize mouse with Alt+3 for PS/2
You should be able to start the debugger with Alt+D (but view only)
Alt+N starts the network
Alt+S shows HDD info


What are the system info shown when in the pre- graphical  scrren?
How much system RAM memory is reported, how much video ram and what is the address of LFB?

It is true that i use only one single method to enable A20 (you should be able to change that from BIOS)
but I never encountered any A20 problems on the PC's is have tested.
