But about using the Visual Studio it self, you just go to the start menu and click on the Visual C++ for example in the program files.
If it's not there you may need to reinstall it.

After you open it you will click on Files->New,
then a dialog opens tell you what type of project you want just click win32 console for basic program and simple one. There you will have to choose between some types of programs. May be the one that could help you to get into the basic thing is the basic "Hello world" program. Select it and then see the code written in C.
