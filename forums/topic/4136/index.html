Lysic,

I do the programming community a service by not releasing the code for qeditor, its a genuine personal nightmare that was never written for public release, it is so complex internally that it would drive people nutz and there is no way I intend to try and support it as released code.

It is also written in a PowerBASIC compiler and is a mixture of asm, basic and C style coding, is post processed and has to have support facilities to construct it.

The TAB setting is an adjustment with the INI editor which you can access from the editor. Just change it to whatever setting you like. There is not much I can do about the font settings as they are passed internally in OLE. It is based on the old version of richedit because its a lot faster that the later one, even though it does not have as many facilities.
