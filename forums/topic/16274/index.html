message box font & color
The MessageBox is a standard Windows dialog and has no built in capacity for font and color except those that are set system wide in the Display Settings. You have to make your own application defined message box and process the WM_CTRLCOLORDLG message, the font can be applied by using the DS_SETFONT style for a DialogEx type dialog.
