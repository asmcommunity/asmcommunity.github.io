Bug in WinASM SDI Program Template
Attention programmers (mostly noobs like myself),

Note that the SDI program template does not include a TranslateMessage() before DispatchMessage() in the program template. If you dont add in the TranslateMessage function undesirable behavior may occur in your window controls. I discovered this while using richedit controls - they will not accept user input unless you use TranslateMessage.
