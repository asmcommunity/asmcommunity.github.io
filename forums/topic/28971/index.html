Re: CPU CACHE PROGRAMMING

Use the below instructions:


      [*]PREFETCHT0
      [*]PREFETCHT1
      [*]PREFETCHT2
      [*]PREFETCHNTA
      [*]MFENCE
      [*]SFENCE
      [*]LFENCE


Good luck. I usually use PREFETCHNTA which proves to be really useful.
