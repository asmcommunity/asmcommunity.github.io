Get window from ID
You can get the handle of any child window provided you assign unique ID's and you know the parent. But GetDlgItem doesn't work with WS_POPUP and WS_OVERLAPPED windows, because the ID's are stored in the same place as menu bar handles. (A bad legacy optimization.)
