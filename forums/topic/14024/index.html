LVS_SORTASCENDING ListView Items Indexes
You can use LVM_SORTITEMSEX and use a callback routine. The sorting routine will send the item numbers for both items to be compared. At that point you can select any subitem that you wish to use as a sortkey. You can also do a custom sort.
