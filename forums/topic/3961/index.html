How to detect virtual CD-ROM
How to detect virtual CD? You don't. If you're going to use it for
software protection, know that your code *will* be tampered.
Complicated stuff like safedisc that uses physical bad sectors and
heavy code encryption are unwrapped faster than you can say
duhwhydidispendmoneyonthiscrap, so using detection to avoid
copying is futile, and only hinders possible legitimate uses (being
able to store all your original games as compressed ISOs on one
of those mean 160gig drives to avoid all the disc swapping, and
possibly increasing speed...)
