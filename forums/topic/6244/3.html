RadASM 1.2.0.0 Released (06-24-2002)
Hi Const.Ex

The ntype member of the DIALOG struct tells you the type.
ntype=0 dialog
ntype=1 edit
ntype=2 static
ntype=3 groupbox

It follows the button index on the dialog design toolbox.
For custom controls you should return control to RadASM. (Return FALSE).
