How to receive the list of devices in Windows
I don't know if it's what you're looking for, but there is a function in the PSAPI dll that enumerates the load addresses of all drivers in the system : EnumDeviceDrivers. There's also a function that return the base name of the driver : GetDeviceDriverBaseName. See PSAPI in the MSDN for more information.
