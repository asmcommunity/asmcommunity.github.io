No task bar icon or address
There are two ways that I know of to accomplish what you want.
The first method is to use WS_EX_TOOLWINDOW style in your window. However, using this style will make your title bar smaller. If this effect is undesirable, try the second method. Create a hidden parent window (ie, no WS_VISIBLE flag and don't call ShowWindow) and then create the window you want to use as its child window.
