Addin for RadASM - Visual Studio
Hi KetilO

I have made an addin starting Visal Studio as a debugger and loading the project executable file where you can step through your code. The path to the application is in a ?SADD? expression on line 109 in ?VSRun.asm?. Improvments could be to involve RadASM.ini to seach for the path. If you add /Zi options to RadASM project debug options dialog, lines ?assemble? and ?asm modul? the exact code is imported to Visal Studio via the *.pdb file. In Visual Studio you can disassembly the code so after for example invoke you can se the op-code for invoke.

Is this something for your IDE.
