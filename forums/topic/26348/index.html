Prevent Windows form shutdowning
Exitwindowsex send a WM_QUERYENDSESSION to all windows, if you return true to this message you will halt the system shutdown. Note if exitwindowsex is called with the EWX_FORCE you will not be able to stop the shutdown. Good luck, Betrayed
