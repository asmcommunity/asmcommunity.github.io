Re: Extremely Basic Help
Here is a general method of compiling.

nasm -f win32 *insert name of the source file with its extension*
gcc -o *insert name of binary*.exe *insert source file with extension here*

I.E:
source file: test.asm
nasm -f win32 test.asm

now it will be an object file: test.obj

gcc -o bin.exe test.obj

Then everything should be nice unless you are running Visat x64.
