Sphinx C-- supporting MS COFF output
Hi friends,

Thanks to the author Michael Sheker, Sphinx C-- can output MS COFF object code. Plus, it's possible to insert other object files to the main project file: 


#include "stringlib.obj"

I attached a small Sphinx C-- MS COFF demo linked with Jeremy Gordon's Golink. You can also use other MS COFF linkers such as MS link and Pelle's Polink.

Golink can be downloaded from:
http://www.jorgon.freeserve.co.uk/Golink.zip
