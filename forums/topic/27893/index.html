QEditor''s Resource Compiler
I asked a while ago about QEditer's resource compiler and if there is a way to stop it from using rsrc.rc. Anyways, all you gotta do is modify the various batch files, substituting %1 for rsrc.rc. Here's the modified bres.bat: 

@echo off
\masm32\bin\rc /v %1.rc
\masm32\bin\cvtres /machine:ix86 %1.res
pause
