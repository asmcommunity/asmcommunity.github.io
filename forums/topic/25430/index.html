HiEdit V1.0.1.3 released
Hi all,

HiEdit control packaged both as a static library and a dll. In the zip you will find the static library, the dll, the header file (HiEdit.inc) and the full source of an application that demonstrates HiEdit control's capabilities.

For those that have not used HiEdit control before:
HiEdit is a multitabbed, ultra fast, large file edit control consuming very little memory. It can display non-printable characters in a readable format and can be used for any general purpose editing of text and birary files.

You can find it on the WinAsm forum (http://www.winasm.net)
