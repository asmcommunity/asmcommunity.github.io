Hi Knighty,

Tough tough problem. I suppose you could use the ITaskbar interface to add an item to the taskbar (I posted code once to do that I think) then get it's bounding rectangle. At that point you would need to subclass the taskbar and snip that region out of the paint routine and paint it yourself. When the user presses your task bar item you will get a WM_SETFOCUS and check the mouse position to find which button was under the cursor on the click. This would be very hard to do though, MS has the source code for Windows so they have a much easier time with cool stuff like that.
