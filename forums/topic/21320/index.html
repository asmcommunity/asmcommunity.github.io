Is there anyway to remove the version info that is automatically included in my project?  I've tried removing its call from the .rc file and aslo deleting the baseVer.rc, and it still comes back regardless of what I do.  I havent looked into it too thouroughly myself but I was wondering if there is a quick way to solve this.  Thanks

Re: Can version info be removed from a project?
The version information is as you suspected in the RC file. If you remove the #include line in the project RC it will not be compiled with the project. However, be sure that you have also disabled the version control addin as it will continue to generate new version numbers.
