RadAsm 2... sniplet window
Hi The Svin

The only way to avoid first file to be selected is to put all files in sub folders or to give it a name higher than first sub folder. BTW Sub folders is also a great way to group them. If you want a file to be selected, do not put it in a sub folder and give it a name lower than first sub folder.

There was a bug in early 2.0.0.x version that would just add text. You should download 2.0.0.7 as there are several important fixes.

Colors in sniplet window will be the same as in code edit window.

Nothing prevents you from having two different versions sharing same ini files. Just rename one of the exe files.
