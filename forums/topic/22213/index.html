Copy all<CTRL+A>+<CTRL+C>
Copy current line<HOME>+<SHIFT+END>+<CTRL+C>
Delete current line<HOME>+<SHIFT+END>+<DEL>
Select current line<HOME>+<SHIFT+END>
Delete to start of line<SHIFT+HOME>+<DEL>
Delete to end of line<SHIFT+END>+<DEL>

Delete to Top<CTRL+SHIFT+HOME>+<DEL>
Copy to Top<CTRL+SHIFT+HOME>+<CTRL+C>
Delete to Bottom<CTRL+SHIFT+END>+<DEL>
Copy to Bottom<CTRL+SHIFT+END>+<CTRL+C>

Select Right Word <CTRL+SHIFT+RIGHTARROW>
Select Left Word <CTRL+SHIFT+LEFTARROW>

works in any editor :)
