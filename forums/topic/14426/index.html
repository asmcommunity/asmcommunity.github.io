BTW peanut,

You must include the dlg's rc file AFTER the menu's rc file i.e.

#include <Res\mainMnu.rc>
#include <Res\mainDlg.rc>

Also you are using a ListView which is a Common Control, it is good practice to InitCommonCtrls > 
include Comctl32.inc

includelib Comctl32.lib



.data

icc	 	QWORD		03FFF00000008h



.code

Error_resource_name_not_found ?
Hi peanut,

You have a dependancy chain. You are using an equate in the Dialog RC that is defined in the Menu.RC so it has to come first. This is normally not a problem unless you enter a Menu Name in the menu dialog (IDR_MENU in your case). If you had just entered 10000 and put 10000 in the menu field of the dialog editor it would probably not have been a problem. Generally I avoid named resources, prefering the numeric only as it has no dependancy chain between resource scripts.
