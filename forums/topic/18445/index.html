Disabling Version Control!
Hi SubEvil,

You can disable it via the menu selection if you want it disabled on a project by project basis or via the Addin Manager if you would like it completely disabled.

To completely disable it just select Addin Manager from the Option menu and find Version Control in the list. Highlight it and press the Uninstall button.

To disable it for a project, select Version Control from the Project menu and remove the check from Enable Version Updating
