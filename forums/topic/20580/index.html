System file not suitable?
Do you have an autoexec.nt file in c:\windows\system32 ? If not, you need to copy one from c:\windows\repair. If a copy does not exist there, you need to get one from your Windows XP CD using expand.exe.

If you do a Google search on 'autoexec.nt' you will find lots of information on this problem.
