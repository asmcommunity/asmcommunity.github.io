muzidowa,

Thanks for publishing these results, qeditor.exe does not write to the registry at all and depending on how the particular machine is set up, it sometimes has the wrong information passed to it by Explorer. It will handle quoted file names, unquoted file names and the variations of Winfile.exe, WinExec, CreateProcess, CMD.EXE and Command.com but it has no way of dealing with the registry as registry dependence is not within its original design criteria.
