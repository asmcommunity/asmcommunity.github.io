Re: em_replacesel...??
Try writing an addin - that will make life much easier for you.

Look at the ResourceID Addin.. The way I do it is to retrieve all text from the window, modify what I want and then write it back using this code


invoke SendMessage,hOpen,EM_EXSETSEL,0,addr chrg
invoke SendMessage,hOpen,EM_GETSELTEXT,0,hMem
; Text is in hMem
invoke SendMessage,hOpen,EM_REPLACESEL,TRUE,hMem
