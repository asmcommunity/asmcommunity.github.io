Include file to import library converter V1.4
Inc2lib converts a POASM \ MASM include file to the associated import library. To create the libraries, the tool launches Pelle's librarian Polib.  inc2lib accepts wildcards like *.inc The libraries can be used with COFF linkers MS link and Pelle's Polink.

The tool offers an optional switch -d to convert include files to module definition files with decorated names. Notice that all the PROTO statements in the include files should be uppercase.
