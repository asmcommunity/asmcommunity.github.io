mistake in PE tutorial?
Hi
In the section table PE tutorial it says that you can use the SizeOfHeaders member of the optional header as an offset to the section table as the section table is after the IMAGE_NT_HEADERS. Well I have found this to not point to the section table and to get to the section table I have to use SIZEOF IMAGE_NT_HEADERS, so I think the value in SizeOfHeaders also includes the section table as well.
