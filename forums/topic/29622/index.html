Re: Meta Data in files
File metadata (for example, the file extension, date etc.) is not stored in file, it's file data that goes inside.

Former stored separately from file content in some other place. For archive this place may be file header, for file system it's some kind of file descriptor/inode (FAT directory entry, e.g.), open file handle can be used in various OS calls to get file metadata.

Please narrow down you question, it's too broad to give you useful pointers.
