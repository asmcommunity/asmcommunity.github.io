Boot CD of sorts
PE Builder is pretty neat program.

Makes a bootable CD of sorts for "rescue" operations.

It is still being improved, but it can:

copy floppies
run 32 bit apps
mspaint

Forgot the other stuff.

If you have 2 CDs, put the boot CD in the first drive.
