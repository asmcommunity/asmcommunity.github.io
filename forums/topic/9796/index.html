AsmEdit V4.7 released (12-26-2002)
Hi all,

I've made many changes to ResEdit and CtrlBuild.

I added six additional controls for both programs.

You can now copy and paste controls.

NOTE! ***********
To move or copy the IPAdress control you need to click on one of the
three dots.

The Animated control is very limited on the type of AVI file you can use.

ResEdit will extract most if not all resources from PE and .res files.
You can extract, display, and save images.
(The program does not extract sound resources yet)

I made several subtle changes to AsmEdit and it's support programs.

I also fixed several nuisance bugs in AsmEdit and it's support programs.

NOTE! You can run ResEdit and CtrlBuild as standalone or as an integrated
function of AsmEdit.

I've upgraded AsmEdit V4.6 to V4.7.

You can go to my Web Site to download the complete package (Masm32V1.zip),
or the service package (SerPk06.zip).

Both are quite large because of the Resource editor, Control builder,
and additional programs.

You can use the service package SerPk06 if you have previously
installed the complete package or SerPk05 dated 10-27-2002.

All of the source code is included.

My Web Site is at:
http://asmedit.massmind.org/

Read the 12-26-2002 section of the Readme.txt file before installing.
Don't forget to run the standalone Setup program after installing.

Check on the Bug Fix download packages, they are updated
as soon as I or someone finds a bug and I fix it.

Thanks,

Ewayne
