AddinHook Macro
mov eax,AddInHook(WM_COMMAND,AIM_TIMER)

the macro returns the hooks value that should be returned from InstallDll. No code is assembled, just a constant value returned given the messages hooked. This allows you to use other instructions like:
   mov eax,AddInHook(WM_COMMAND)

   or aTimer,0

   je @F

   or eax,AddInHook(AIM_TIMER)
