Not a homework (bignum a2dw)
Pseudo-Code:
Number <-- 0

Index <-- Start of string



If String(Index) = "-" Then

    Negitive <-- True

    Index <-- Index + 1

Else

    Negitive <-- False

End If



While Not Index = End of string

    Temp <-- String(Index) - "0"

    Number <-- Number * 10 + Temp

    Index <-- Index + 1

End While



If Negitive Then

    Number <-- Number * -1

End If
No error checking is done.
