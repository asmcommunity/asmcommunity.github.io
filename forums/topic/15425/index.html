WinAsm V2.0.0.0
Hi all,

This is WinAsm V2.0.0.0

What's new
1. Project Explorer and Out Parent are now Docking/float windows. (Thanks Maco, Manos, eko)

Notes:
a).Custom title bar styles; Right-click on the non client area and choose the title style from the context menu.
b).Pressing Ctrl while moving prevents docking (Always float)
c).Double clicking on a docked window converts it to float (last float pos and size)

2.Quotes for /Out switch so that project can be built even when there are spaces in filename/folder - (Thanks SOL_bl, andrew_k).

3.Exernal tools are now executed with ShellExecute (Thanks P2M)

4.Minimum height and width of main window set to 220 pixels.

5.Print Dialog Screen center (Thanks andrew_k).

6.Improvement on how all windows receive/loose focus

7.Image for Blocks Tab Item.

8.Esc hides the output window (Thanks SOL_bl)

9.Much more comprehensive MasmApiCall.vaa (Thanks PhoBos). The previous renamed to MasmApiCallLt.vaa for those who might have startup speed problems.

10.Some more bitmapped menu items and CodeHi different images (Thanks andrew_k for the images).

Regards,

akyprian
