If it's a corrupt filesystem, the best you can do is PLUG OUT THE DISK... (well, dont' run scandisk or whatever, and if you have pagefile/temporary files, hurry and relocate those to another drive). Then, get some recovery/forensic tools - like restorer2000 pro, which usually does a pretty good job. Be sure to restore to another drive.

Oh, and better switch to NTFS, it's way more robust than FAT - even with device-driver and hardware-caused BSODs, I've not had a single file corruption for... bum bum... well, since I installed win2k anyway, which has been some years already.
