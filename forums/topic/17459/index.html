How to produce other files ? (need help with switches option
Hi Neitsa

Use RadASM's menu Project / Project Options and set Assembler to:

3,O,$B\ML.EXE /c /coff /Cp /nologo /Zi /Fl /Fm /I"$I",2

To make this default, edit masm.ini
