Generic splash screen library
Exactly what the title says... it's a source code, very easy to edit to suit your needs. Just change the values of some equates in the main file, change the file names in the resource script, and rebuild it. Piece of cake :)
It supports both BITMAP and IMAGE resource types, and also a custom shape (for transparencies) using regions.
Have fun ;)
