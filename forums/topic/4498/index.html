MACRO: converting integer to string
How do I convert a number to a string in a macro? I wanna do something like this:

nothing MACRO Name
number = 8
call &Name&number
ENDM
Posted on 2002-03-29 01:43:16 by marcinbu
MACRO: converting integer to string
nothing MACRO Name

	number = 8

	call @CatStr(&Name,%number)

ENDM
