Getting actions if a bitmap (picurebox) is clicked
Do you mean get the handle of some item on the Dialog? If so, then yes it is possible like so: 

invoke GetDlgItem, hWnd, ID_OF_OBJECT
will return the handle of the object refered to by ID_OF_OBJECT, eg. In rsrc.rc 

#define IDB_EN    3000
....
PUSHBUTTON "My Button", IDB_EN, 25, 5, 90, 13
....
In .ASM file 

.const
IDB_EN    equ     3000
...
...
invoke GetDlgItem, hWnd, IDB_EN
mov hButton, eax
...
Where hWnd is the handle of your proc! Mirno
