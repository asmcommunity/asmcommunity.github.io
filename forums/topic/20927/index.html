Re: Problem / Bug with menu 'Set Paths'
Hi

A better way is to update ini files, templates and your projects with quotes.

For ini files (all project types):
7=0,0,"$E\OllyDbg",$.exe
17=0,0,"$E\OllyDbg",$.exe

For projects:
Run w/Debug: 0,0,"$E\OllyDbg",5
