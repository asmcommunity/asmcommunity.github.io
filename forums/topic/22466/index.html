Welcome!

I use RadASM and FASM exclusively for my programming needs.  You do not need the FASM.CT file to use these two fine products.

The .CT files are just code templates which can allow you to design the basic structure for different ASM file type:

plain Windows
Dialog
Dialog as Window
Console
...

Without the template files, you simply have to do all the typing yourself!  As soon as I get some free time, next year, I'm going to construct ones for my own needs and programming style.

Re: Spanish newbie need help
{push regs}
push ebx edi esi
~
pop edi esi ebx
{anything you want}
xor eax,eax
.1
cmp eax,~
jnz .1
{}


copy this and save it as fasm.ct in radasm addins directory
now open some radasm fasm project and press Ctrl+J
now you get option to insert that snippet
this is just to show you how it works,
you can add anything you want, thats the purpose of it,
to insert the code blocks you frequently use...

i hope this helped. you can read the addin description by going to menu:
Options->Addin Manager, select CodeTemplates, click Help button
