File I/O
It will look like: "AABBB"

If the file has "AABAA" and you want to delete the B - you need to copy it manually skip the B. Like:

lea eax, BufferToFile
add eax, OffsetToLetterToCutOut (like 3)
.WHILE byte ptr != 0
mov cl, byte ptr
mov byte ptr , cl
.ENDW
