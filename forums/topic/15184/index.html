why dosent this work
In your code, AX is the LO_WORD of the wParam and would contain the ID of the control.

If you assigned the value of 1 as the ID of your rich edit control, your code should work. If the ID of your rich edit control is NOT 1, try using:
.if ax==proper ID value of rich edit control
