Measuring clocks in win
Windows actualy does allow you to do that using the functions 'SetPriotyClass', and 'SetThreadPrioty'. A much shorter alternitive under Win9x is to use a CLI instruction. This disables interupts preventing windows from gaining control of the CPU completly. You can re-enable interupts with the STI instruction.
