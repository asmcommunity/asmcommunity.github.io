Re: looking for some registry entries from XP SP1
I assume you have lost them?
You can fix this easily by booting from the XP installation cd, select repair, choose console, and rename c:\windows\system32\config\system to c:\windows\system32\config\system.old
after that reboot and you should have a fresh Hardware profile again.

or use the safer approach by copying the system.bak file over the one you have now.
There is no need to copy the other files as well, even though this article says so.
