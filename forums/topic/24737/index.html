RAVideo, a simple video player custom control
RAVideo is a very easy to use video player custom control for?  RadASM.

Copy RAVideo.dll to C:\RadASM or to C:\Windows\System
Activate it in RadASM using Options / Custom Controls

Messages:
RAVID_PLAY wParam=0, lParam=lpFileName
RAVID_STOP wParam=0, lParam=0

Styles:
STYLE_PLAYBAR Shows the standard playbar.
STYLE_KEEPASPECT Keeps the aspect ratio when window is sized.
