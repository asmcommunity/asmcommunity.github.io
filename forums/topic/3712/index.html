Interesting finding concerning compiling and linking
Not sure if everybody nows this but link
and ML can take parameters that are set in in
the environment.

so one can add this line to their environment
set ml=/c /coff /Cp /I:pathtoinclude
and then to assemble a file
all one has to do type is
ML file.asm
You can also do this with link also
eg
set link=/SUBSYSTEM:WINDOWS /MACHINE:X86 file.res
and then just type link file.obj and these params will be expanded to link.

Anyways i was happy to figure this out, hope
this helps somebody else.
