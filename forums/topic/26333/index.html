Printing small amount of ASCII text
If you want to print the text onto the window you have to get the dc handle and textout or drawtext, or you can use the static or edit control which draw the text automatically, you can use the richtext control for more features (bold, italic..). Look up whatever one you need in the win32 programming reference or you can download a few examples of all of those on this site.
