Re: RadAsm ObjAsm addin question
Hi Rags
The menu choices in the RadASM AddIn serve for code optimization.
The first one (Method locals use) checks for unused declared locals within a method.
The second (Method register preservation) checks for the ebx, edi and esi registers if they are declared with the “uses??? keyword and not used in a method or used and not declared.

In all situations, a notification message is sent to RadASM’s secondary output window indication the severity of finding. Double clicking on the first line of each message moves the cursor to the error line within the source code.
