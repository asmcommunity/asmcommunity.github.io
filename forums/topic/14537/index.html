Well i would call that Basic Line Subdivision: what happens when you reach the areas of high curvature? The length of the line sections needs to far shorter in these areas, and longer in the low curvature areas to get an optimum line approximation of the curve.

This does not need to be real time, but will have to be executed as preprocessing, saving it to a vertex buffer and then using that at run time.
