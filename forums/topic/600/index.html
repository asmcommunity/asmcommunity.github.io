Asm code type ahead (AsmTypeA) Update
I've made a change to the TypeAhead routine when
the Auto Completion is turned off you can now
acknowledge the generated field by either
pressing the Space bar or Tab key.

I still like the Auto Completion turned on, it's
so much faster and if there is a conflict with
the Keyword and a label or varible all you have
to do is press the undo button and type over it.

Download the latest program.

You enter everything in lower case and from
column one.

Turn the Auto Completion off and enter the
following:

INVOKE SendMessage, hTree, TVM_SORTCHILDRENCB, addr TV_SORTCB

Then turn the Auto Completion on and then enter
the following:

1. 1m (1 is a one)
2. hTree, TVM_SORTCHILDRENCB, addr TV_SORTCB

Then enter:

1. 1cw (1 is a one)
