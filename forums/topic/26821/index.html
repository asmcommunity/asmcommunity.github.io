Searching for good Resource Editor
I don't think there has been a GOOD resource editor for a long time, the VC6 internal one has a good range of controls but it will only use the VC6 include files and risks messing up any external version of resource.h. Its also buried in a big slow messy looking IDE so I never started using it. My own choice of a VERY poor lot is the old Symantec version. Its a bit buggy here and there but it has the good habit of not messing up an existing RC script with C++ junk. It does most of the available controls, its icon and bitmap editors work OK, dialog editor is OK if you ONLY use numeric IDs. I have a copy on my site under the name NRE.EXE at URL, http://www.pbq.com.au/home/hutch/tools.htm It may do the job for you OK. Regards, hutch@pbq.com.au
