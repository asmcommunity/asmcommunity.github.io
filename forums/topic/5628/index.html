Noob question
you can't do memory to memory copies in asm. You must do

mov , nID
mov mii.wID,

so if the eax register is free,

mov eax, nID
mov mii.wID, eax

will work. Or if you don't have any free registers to use then you can push & pop the values:

push nID
pop mii.wID
