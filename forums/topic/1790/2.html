Loading constants (FPU)
If I were writing the program, I would certainly opt for pushing the address of the REAL8 as the simpler solution for passing a parameter, and have the function retrieve it directly from global memory.

If I had no choice due to using someone else's function requiring the REAL8 as the parameter on the stack, I would still have to store it in a memory location of my own and then push it one dword at a time.
