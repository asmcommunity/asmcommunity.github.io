A simple setup guide
Hi evenings

If you have tasm installed to C:\Tasm and RadASM installed to C:\RadASM then no setup is needed.
You can however make tasm the default assembler:

1. Open RadASM.ini
2. Change the following:


Assembler=masm,fasm,tasm,nasm,hla,GoAsm

to:


Assembler=tasm,masm,fasm,nasm,hla,GoAsm

To fully take advantage of RadASM's features you must create projects from your files.

RadASM does not use make by default.
