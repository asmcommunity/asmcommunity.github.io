RadAsm+LIB Project (for who interest)
I could not used RadAsm as LIB project because of... (fill in the banks whom tried to do this job). So i tried to implement this with a simple utility which creates a response file. At last this changes needs in Tasm.ini
;;;;;;;;;;



Type=Win32 App,LIB Project <<-- add this last part.

<<-- add this section after section

Files=1,1,0,1,0
Folders=1,0,0
MenuMake=0,0,1,0,0,0,0,1
;x=MakeFileToDelete/CheckExistsOnExit,
;(O)utput/(C)onsole/0,Command,
;MakeFile1[,MakeFile2[,MakeFile3...]]
1=
2=
3=9,O,$B\TLIB.EXE $9 /C /E @Objes.rsp
4=0,0,,5
5=
6=*.obj,O,$B\tasm32.exe /ml /m4 /z /q /zn /I"$I",*.asm
11=
12=
13=9,O,$B\TLIB.EXE $9 /C /E @Objes.rsp
14=0,0,,5
15=
16=*.obj,O,$B\tasm32.EXE /ml /m4 /z /q /zi /I"$I",*.asm

;;;;;;;;;;;

Here you must put the objes.exe in c:\RadAsm directory then add this executable in "Tools" menu in RadAsm. (I couldn't used this in "Make menu").
If you want to create lib follow these sequence,

1. Assemble All Modules
2. Create Response File (simply run from menu)
3. Link

Note: Must to add .asm files as "modules", NOT as "File".
