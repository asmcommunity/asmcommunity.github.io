One way is with OpenProcess or CreateProcessEx and GetExitCodeProcess. It can be used in a busy wait.

Another way is to use WaitForSingleObject, but that's not recommended unless you use it in a thread that doesn't involve graphics. Otherwise, it will delay window events and updates.
