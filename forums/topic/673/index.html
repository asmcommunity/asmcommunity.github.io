Secondary window
hello,

now to create a second window just do the same as when creating your first window, that is , use CreateWindowEx(). If the second window shall act different, it will also require its own "window class" (that is, call RegisterClass() for your second window and code a second window proc). If you want your second window having its own message loop (but why should you?), you need to create a second thread for it.
