Debug info
karim, I use VC++ to debug - this is the custom build line I use :- 

\masm32\bin\ml.exe /nologo /coff /Zi /Zd /Zf  /c  $(InputName).asm
I use the VC++ Linker, not the one supplied with the MASM32 package, as you have to use that in debug mode also. Look at this message for a full discussion of how I make the VC++ IDE into a MASM IDE :) umbongo.
