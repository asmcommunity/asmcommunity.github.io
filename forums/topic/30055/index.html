Pixelmap UPDATE
Hello!

Attached is an update for ObjAsm32's Pixelmap object.
Three new methods have been added:

PrintAt - Draws text to the pixelmap using any given Font Handle
Circle - Plots a 2D Circle, given origin, radius and color
Line - Plots a 2D Line between any two given points, using the given color

I think it's tragic that Pixelmap has high-end functionality like gamma-ramping, yet is missing such basic functions as drawing lines ... I believe these new functions greatly increase the usefulness of the class.
