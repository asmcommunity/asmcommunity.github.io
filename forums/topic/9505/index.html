Can you add FFREEP to FASM?
I have added to my version - it is a documented AMD
instruction and works on all Pentium MMX or greater.

Thank you.

FFREEP st(i) = DF C0+i

For example:

FFREEP st(1)

would assemble to:

db 0DFh, 0C1h
