interface viewer for OLEView in ASM
I've implemented OLEView's interface viewer "API" in comview (version 1.7.x only, which isnt finished yet) and written such a viewer in ASM, for interface IShellFolder. If interested, download it from here . Currently you may use it with OLEView only.

It shows the items in the folder and allows to open a new window for subfolders. Nothing too exciting, but may be improved and will possibly serve as sample code in next comview version.
