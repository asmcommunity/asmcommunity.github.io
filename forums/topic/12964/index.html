like it were a bitmap
That is probably the easiest route to take if you are already familiar with that notion.

First create a compatible DC, then a bitmap of the required size which you select into the compatible DC and use as your memory array.

When a user clicks a mouse button, you retrieve the mouse's coordinates and map it to the location of your grid. Copy changes you make in the compatible DC to the screen's DC.
