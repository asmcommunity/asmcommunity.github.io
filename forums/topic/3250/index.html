How to: Use RadASM over a network
It's possible to set up RadASM so you can use it from any pc on your network without installing any files on it. Just edit the appropriate assembler init file. For masm.ini:


$A=\\Mypc\C\MASM32
$B=$A\Bin
$D=\\Mypc\C\RadASM\AddIns
$H=$A\Help
$I=$A\Include
$L=$A\Lib
$P=\\Mypc\C\RadASM\Projects
$S=\\Mypc\C\RadASM\Sniplets
$T=\\Mypc\C\RadASM\Templates

Just change \\Mypc\C\ to the correct computer & share name.
This will make it work on both the local pc and any network pc's without using drive mappings or anything.
