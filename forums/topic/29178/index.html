ObjAsm32 1.4b released
Today I want to announce the release of our newest ObjAsm32 release package.

It contains all the changes and improvements we were working the last months, ranging from model improvements to new DirectX projects.
The list of changes is quite extensive and can be found in the Documentation page of our homepage mirrors, which were updated recently.

We think that our OOP model has reached a good level of maturity and due the fact that we are recently coding for commercial projects that we are considering freezing this state.
That means that we will not update the project so frequently as before but with the new developments which will grow from those projects.

We hope you enjoy our work!

http://www.objasm32.rantx.com/
