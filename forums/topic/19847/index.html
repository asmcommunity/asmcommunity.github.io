There was never posted a reply to this post. Possibly because no one have run-into this problem before. So therefore i'll post the solution i found to work. It was so simple that I was kinda embarrased by the fact that i didnt read the documentation good enough. Anyways, here is the line of code you'll need to solve the problem. For a description you can look it up in MSDN. 

inv   SendMessage, [g_hListView], LVM_SETICONSPACING, 0, 00XX00XXH
