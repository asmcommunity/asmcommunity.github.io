NASM 2.07rc4 - Bringing 64-bit Mach-O (macho64) Support
As the subject says, NASM 2.07rc4 is out.

This release candidate primarily marks the official support for the 64-
bit Mach-O object format, i.e. macho64.

This release candidate also improves upon regular 32-bit Mach-O (macho/
macho32) support, among other routine improvements/bug-fixes.

If you have an interest in 64-bit Mac OS X ASM development, feel free
to give this release candidate a thorough shakedown.
