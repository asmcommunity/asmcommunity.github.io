Its possible, but you will need to dig up detailled documentation on the interfaces. However, this project will be alot of work to do though! You will need to generate headers for MASM, then write code to establish COM objects to operate the OCX as required. You will need to learn the methology of COM and specifically how OCX's fall into this picture.

If your seriously willing to invest the overhead in learning, i suggest taking this project to the COM section, where you might get the attention of additional people skilled in these areas.
